# 模态入门

## 注册

通过 https://modal.com 免费注册并获得每月 30 美元的积分。

## 身份验证

使用 Modal CLI 设置身份验证：

```bash
modal token new
```

这将在 `~/.modal.toml` 中创建凭据。或者，设置环境变量：
- `MODAL_TOKEN_ID`
- `MODAL_TOKEN_SECRET`

## 基本概念

### Modal 是无服务器的

Modal 是一个无服务器平台 - 只需为使用的资源付费，并在几秒钟内按需启动容器。

### 核心组件

**App**：表示运行在Modal上的应用程序，将一个或多个Function分组进行原子部署。

**功能**：作为独立单元，独立缩放。当没有实时输入时，不会运行任何容器（并且不收费）。

**镜像**：环境代码在安装了依赖项的容器快照中运行。

## 主要特点

- **快速原型制作**：编写 Python，几秒钟内在 GPU 上运行
- **无服务器 API**：使用装饰器创建 Web 端点
- **计划作业**：在云中运行 cron 作业
- **GPU 推理**：访问 T4、L4、A10、A100、H100、H200、B200 GPU
- **分布式卷**：机器学习模型的持久存储
- **沙箱**：不受信任代码的安全容器
