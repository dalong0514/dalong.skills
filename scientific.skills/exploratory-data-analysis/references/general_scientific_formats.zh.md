# 通用科学数据格式参考

本参考涵盖跨多个学科使用的通用科学数据格式。

## 数值和数组数据

### .npy - NumPy 数组
**描述：** 二进制 NumPy 数组格式
**典型数据：** 任何数据类型的 N 维数组
**用例：** 用于数值数据、中间结果的快速 I/O
**Python 库：**
- `numpy`：`np.load('file.npy')`、`np.save()`
- 内存映射访问：`np.load('file.npy', mmap_mode='r')`
**EDA 方法：**
- 数组形状和维数
- 数据类型和精度
- 统计摘要（平均值、标准差、最小值、最大值、百分位数）
- 缺失值或无效值（NaN、inf）
- 内存占用
- 价值分布和直方图
- 稀疏性分析
- 相关结构（如果是二维）

### .npz - 压缩的 NumPy 存档
**描述：** 一个文件中的多个 NumPy 数组
**典型数据：**相关数组的集合
**用例：** 将多个数组保存在一起，压缩存储
**Python 库：**
- `numpy`: `np.load('file.npz')` 返回类似字典的对象
- `np.savez()` 或 `np.savez_compressed()`
**EDA 方法：**
- 包含的数组列表
- 个体阵列分析
- 数组之间的关系
- 文件总大小和压缩率
- 命名约定
- 数据一致性检查

### .csv - 逗号分隔值
**描述：**纯文本表格数据
**典型数据：** 实验测量、结果表
**用例：** 通用数据交换、电子表格导出
**Python 库：**
- `pandas`: `pd.read_csv('file.csv')`
- `csv`：内置模块
- `polars`：高性能 CSV 读取
- `numpy`：`np.loadtxt()` 或 `np.genfromtxt()`
**EDA 方法：**
- 行数和列数
- 数据类型推断
- 缺失值模式和频率
- 列统计（数字：平均值、标准差；分类：频率）
- 异常值检测
- 相关矩阵
- 重复行检测
- 标头和索引验证
- 编码问题检测

### .tsv / .tab - 制表符分隔值
**描述：**制表符分隔的表格数据
**典型数据：** 与 CSV 类似，但以制表符分隔
**用例：**生物信息学、文本处理输出
**Python 库：**
- `pandas`：`pd.read_csv('file.tsv', sep='\t')`
**EDA 方法：**
- 与 CSV 格式相同
- 制表符与空格验证
- 报价处理

### .xlsx / .xls - Excel 电子表格
**描述：** Microsoft Excel 二进制/XML 格式
**典型数据：** 带有格式、公式的表格数据
**用例：** 实验室笔记本、数据输入、报告
**Python 库：**
- `pandas`：`pd.read_excel('file.xlsx')`
- `openpyxl`：完整的 Excel 文件操作
- `xlrd`：读取 .xls（旧版）
**EDA 方法：**
- 工作表枚举和名称
- 每张数据分析
- 配方评估
- 合并单元格处理
- 隐藏行/列
- 数据验证规则
- 命名范围
- 仅格式化单元格检测

### .json - JavaScript 对象表示法
**描述：** 分层文本数据格式
**典型数据：** 嵌套数据结构、元数据
**用例：** API 响应、配置、结果
**Python 库：**
- `json`：内置模块
- `pandas`：`pd.read_json()`
- `ujson`：更快的 JSON 解析
**EDA 方法：**
- 模式推断
- 嵌套深度
- 键值分布
- 数组长度
- 数据类型一致性
- 钥匙丢失
- 重复检测
- 规模和复杂性指标

### .xml - 可扩展标记语言
**描述：** 分层标记格式
**典型数据：** 带有元数据的结构化数据
**用例：** 基于标准的数据交换、API
**Python 库：**
- `lxml`：`lxml.etree.parse()`
- `xml.etree.ElementTree`：内置 XML
- `xmltodict`：将 XML 转换为字典
**EDA 方法：**
- 模式/DTD 验证
- 元素层次结构和深度
- 命名空间处理
- 属性与元素内容
- CDATA 部分
- 文本内容提取
- 兄弟姐妹和孩子的数量

### .yaml / .yml - YAML
**描述：** 人类可读的数据序列化
**典型数据：** 配置、元数据、参数
**用例：** 实验配置、管道
**Python 库：**
- `yaml`：`yaml.safe_load()` 或 `yaml.load()`
- `ruamel.yaml`：YAML 1.2 支持
**EDA 方法：**
- 配置结构
- 数据类型处理
- 列表和字典深度
- 锚点和别名的使用
- 多文档文件
- 评论保存
- 根据模式进行验证
### .toml - TOML 配置
**说明：** 配置文件格式
**典型数据：** 设置、参数
**用例：** Python 包配置、设置
**Python 库：**
- `tomli` / `tomllib`：TOML 读取（Python 3.11+ 中的 tomllib）
- `toml`：读取和写入
**EDA 方法：**
- 剖面结构
- 键值对
- 数据类型推断
- 嵌套表验证
- 必填字段与可选字段

### .ini - INI 配置
**说明：**简单的配置格式
**典型数据：** 应用程序设置
**用例：** 传统配置，简单设置
**Python 库：**
- `configparser`：内置 INI 解析器
**EDA 方法：**
- 节枚举
- 键值提取
- 类型转换
- 评论处理
- 区分大小写

**典型数据：** 多维数组、元数据、组

**EDA 方法：**
- 组和数据集层次结构
- 数据集形状和数据类型
- 属性和元数据
- 压缩和分块策略
- 内存高效采样
- 数据集关系
- 文件大小和效率
- 访问模式优化

### .zarr - 分块数组存储
**描述：**云优化分块数组
**典型数据：** 大型 N 维数组
**用例：**云存储、并行计算、流媒体
**Python 库：**
- `zarr`：`zarr.open('file.zarr')`
- `xarray`：Zarr 后端支持
**EDA 方法：**
- 数组元数据和维度
- 块大小优化
- 压缩编解码器和比率
- 同步器和存储类型
- 多尺度层次结构
- 并行访问性能
- 属性元数据

### .gz / .gzip - Gzip 压缩
**描述：** 压缩数据文件
**典型数据：**任何压缩文本或二进制
**用例：** 存储/传输压缩
**Python 库：**
- `gzip`：内置gzip模块
- `pandas`：读取函数中的自动 gzip 处理
**EDA 方法：**
- 压缩比
- 原始文件类型检测
- 减压验证
- 标题信息
- 多成员档案

### .bz2 - Bzip2 压缩
**描述：** Bzip2 压缩
**典型数据：** 高度压缩的文件
**用例：** 比 gzip 更好的压缩
**Python 库：**
- `bz2`：内置 bz2 模块

**EDA 方法：**
- 压缩效率

### .zip - ZIP 存档

- 存档会员列表

- 目录结构

### .tar / .tar.gz - TAR 存档

- `tarfile`：内置 TAR 支持

- 会员档案列表

## 时间序列和波形数据

### .wav - 波形音频
**描述：** 音频波形数据
**典型数据：** 声音信号、录音
**用例：** 声学分析、超声波、信号处理
**Python 库：**
- `scipy.io.wavfile`：`scipy.io.wavfile.read()`
- `wave`：内置模块
- `soundfile`：增强音频 I/O
**EDA 方法：**
- 采样率和持续时间
- 位深度和通道
- 幅度分布
- 频谱分析（FFT）
- 信噪比
- 削波检测
- 频率内容

### .mat - MATLAB 数据
**描述：** MATLAB 工作区变量
**典型数据：** 数组、结构体、单元格
**用例：** MATLAB-Python 互操作性
**Python 库：**
- `scipy.io`：`scipy.io.loadmat()`
- `h5py`：对于 MATLAB v7.3 文件（基于 HDF5）
- `mat73`：v7.3 的纯 Python
**EDA 方法：**
- 变量名称和类型
- 数组维度
- 构造领域勘探
- 单元阵列处理
- 稀疏矩阵检测
- MATLAB版本兼容性
- 元数据提取

### .edf - 欧洲数据格式
**描述：**时间序列数据（尤其是医疗）
**典型数据：** 脑电图、生理信号
**用例：** 医疗信号存储
**Python 库：**
- `pyedflib`：EDF/EDF+ 读写
- `mne`：神经生理学数据（支持 EDF）
**EDA 方法：**
- 信号计数和名称
- 采样频率
- 信号范围和单位
- 录音时长
- 注释事件
- 数据质量（饱和度、噪声）
- 患者/研究信息

### .csv（时间序列）
**描述：** 带时间戳列的 CSV
**典型数据：** 时间索引测量值
**用例：** 传感器数据、监控、实验
**Python 库：**
- `pandas`：`pd.read_csv()` 与 `parse_dates`
**EDA 方法：**
- 时间范围和分辨率
- 采样规律性
- 缺少时间点
- 趋势和季节性
- 平稳性测试
- 自相关
- 异常检测

## 地理空间和环境数据

### .shp - 形状文件
**描述：** 地理空间矢量数据
**典型数据：** 地理特征（点、线、多边形）
**用例：** GIS 分析、空间数据
**Python 库：**
- `geopandas`：`gpd.read_file('file.shp')`
- `fiona`：较低级别的 shapefile 访问
- `pyshp`：纯Python形状文件阅读器
**EDA 方法：**
- 几何类型和数量
- 坐标参考系统
- 边界框
- 属性表分析
- 几何有效性
- 空间分布
- 多部分功能
- 关联文件（.shx、.dbf、.prj）

### .geojson - GeoJSON
**描述：** 地理数据的 JSON 格式
**典型数据：** 具有几何形状和属性的特征
**用例：** 网络地图、空间分析
**Python 库：**
- `geopandas`：原生 GeoJSON 支持
- `json`：解析为 JSON 然后处理
**EDA 方法：**
- 功能数量和类型
- CRS规范
- 边界框计算
- 属性架构
- 几何复杂性
- 嵌套结构

### .tif / .tiff（地理空间）
**描述：** 具有空间参考的 GeoTIFF
**典型数据：** 卫星图像、DEM、栅格
**用例：** 遥感、地形分析
**Python 库：**
- `rasterio`：`rasterio.open('file.tif')`
- `gdal`：地理空间数据抽象库
- `xarray` 与 `rioxarray`：N 维地理空间数组
**EDA 方法：**
- 光栅尺寸和分辨率
- 乐队数量和描述
- 坐标参考系
- 地理变换参数
- NoData 值处理
- 像素值分布
- 直方图分析
- 概述和金字塔

### .nc / .netcdf - 网络通用数据格式
**描述：** 自描述的基于数组的数据
**典型数据：** 气候、大气、海洋数据
**用例：**科学数据集、模型输出
**Python 库：**
- `netCDF4`：`netCDF4.Dataset('file.nc')`
- `xarray`：`xr.open_dataset('file.nc')`
**EDA 方法：**
- 变量枚举
- 维度分析
- 时间序列属性
- 空间覆盖
- 属性元数据（CF约定）
- 坐标系
- 分块和压缩
- 数据质量标志

### .grib / .grib2 - 网格二进制
**说明：**气象数据格式
**典型数据：** 天气预报、气候数据
**用例：** 数值天气预报
**Python 库：**
- `pygrib`：GRIB文件读取
- `xarray` 与 `cfgrib`：GRIB 到 xarray
**EDA 方法：**
- 消息库存
- 参数和级别类型
- 空间网格规范
- 时间覆盖范围
- 乐团成员
- 预测与分析
- 数据打包和精度

### .hdf4 - HDF4 格式
**描述：** 较旧的 HDF 格式
**典型数据：** NASA 地球科学数据
**用例：** 卫星数据（MODIS 等）
**Python 库：**
- `pyhdf`：HDF4 访问
- `gdal`：可以读取 HDF4
**EDA 方法：**
- 科学数据集列表
- Vdata和属性
- 尺寸尺度
- 元数据提取
- 质量标志
- 转换为 HDF5 或 NetCDF

## 专门的科学格式

### .fits - 灵活的图像传输系统
**描述：** 天文数据格式
**典型数据：** 望远镜的图像、表格、光谱

- 图像尺寸和WCS
- 标题关键词分析
- 表格列说明

- FITS 公约合规性
- 校验和验证

**典型数据：**复杂的分层科学数据

- `asdf`：`asdf.open('file.asdf')`
**EDA 方法：**
- 树结构探索
- 模式验证

- 压缩方法
- YAML 元数据
- 版本兼容性

### .root - ROOT 数据格式

- `uproot`：纯Python ROOT阅读
- `ROOT`：官方 PyROOT 绑定
**EDA 方法：**
- TTree结构
- 分支类型和条目
- 直方图库存

### .txt - 纯文本数据
**描述：** 基于文本的通用数据
