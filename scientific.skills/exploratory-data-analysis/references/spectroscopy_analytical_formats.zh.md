# 光谱学和分析化学文件格式参考

本参考涵盖各种光谱技术和分析化学仪器中使用的文件格式。

## 核磁共振波谱

### .fid - 无核磁共振感应衰变
**描述：** 来自 Bruker、Agilent、JEOL 的原始时域 NMR 数据
**典型数据：** 复杂时域信号
**用例：** NMR 波谱、结构解析
**Python 库：**
- `nmrglue`：`nmrglue.bruker.read_fid('fid')` 或 `nmrglue.varian.read_fid('fid')`
- `nmrstarlib`：NMR 数据处理
**EDA 方法：**
- 时域信号衰减
- 采样率和采集时间
- 数据点数量
- 信噪比估计
- 基线漂移评估
- 数字滤镜效果
- 采集参数验证
- 变迹函数选择

### .ft / .ft1 / .ft2 - NMR 频域
**描述：** 傅里叶变换核磁共振谱
**典型数据：** 处理后的频域数据
**用例：** NMR 分析、峰积分
**Python 库：**
- `nmrglue`：频域读取
- 定制加工管道
**EDA 方法：**
- 峰值选取和整合
- 化学位移范围
- 基线校正质量
- 相位校正评估
- 参考峰识别
- 光谱分辨率
- 文物检测
- 多重性分析

### .1r / .2rr - 布鲁克 NMR 处理数据
**描述：**布鲁克处理过的光谱（实部）
**典型数据：** 1D 或 2D 处理的 NMR 谱
**用例：** 使用布鲁克软件进行 NMR 数据分析
**Python 库：**
- `nmrglue`：布鲁克格式支持
**EDA 方法：**
- 加工参数审查
- 窗函数效果
- 零填充评估
- 线性预测验证
- 光谱伪影

### .dx - NMR JCAMP-DX
**描述：** 用于 NMR 的 JCAMP-DX 格式
**典型数据：**标准化核磁共振谱
**用例：** 软件之间的数据交换
**Python 库：**
- `jcamp`：JCAMP 阅读器
- `nmrglue`：可以导入JCAMP
**EDA 方法：**
- 格式合规性
- 元数据完整性
- 峰值表验证
- 积分值
- 化合物识别信息

### .mnova - Mnova 格式
**描述：** Mestrelab Research Mnova 格式
**典型数据：** 带有处理信息的 NMR 数据
**用例：** Mnova 软件工作流程
**Python 库：**
- `nmrglue`：有限的 Mnova 支持
- 标准格式的转换工具
**EDA 方法：**
- 多光谱处理
- 处理流程审查
- 量化数据
- 结构分配

## 质谱分析

### .mzML - 质谱标记语言
**描述：** 基于 XML 的标准 MS 格式
**典型数据：** MS 谱图、色谱图、元数据
**用例：**蛋白质组学、代谢组学、脂质组学
**Python 库：**
- `pymzml`：`pymzml.run.Reader('file.mzML')`
- `pyteomics.mzml`：`pyteomics.mzml.read('file.mzML')`
- `MSFileReader`：各种包装器
**EDA 方法：**
- 扫描计数和 MS 水平分布
- 保留时间范围和 TIC
- m/z 范围和分辨率
- 母离子选择
- 碎片模式
- 仪器配置
- 质量控制指标
- 数据完整性

### .mzXML - 质谱 XML
**描述：** 旧版 XML MS 格式
**典型数据：** 质谱和色谱图
**用例：** 蛋白质组学工作流程（较旧）
**Python 库：**
- `pyteomics.mzxml`
- `pymzml`：可以读取 mzXML
**EDA 方法：**
- 类似于mzML
- 版本兼容性
- 转换质量评估
