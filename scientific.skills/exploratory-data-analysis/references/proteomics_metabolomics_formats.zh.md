# 蛋白质组学和代谢组学文件格式参考

该参考文献涵盖了蛋白质组学、代谢组学、脂质组学和相关组学工作流程特有的文件格式。

## 基于质谱的蛋白质组学

### .mzML - 质谱标记语言
**描述：** MS 数据的标准 XML 格式
**典型数据：** MS1 和 MS2 谱图、保留时间、强度
**用例：**蛋白质组学、代谢组学流程
**Python 库：**
- `pymzml`：`pymzml.run.Reader('file.mzML')`
- `pyteomics.mzml`：`pyteomics.mzml.read('file.mzML')`
- `pyopenms`：OpenMS Python 绑定
**EDA 方法：**
- 扫描计数和 MS 水平分布
- 总离子色谱（TIC）分析
- 基峰色谱图（BPC）
- m/z 覆盖范围和分辨率
- 保留时间范围
- 前体选择模式
- 数据完整性
- 质量控制指标（锁定质量、标准）

### .mzXML - 旧版 MS XML 格式
**描述：** 较旧的基于 XML 的 MS 格式
**典型数据：** 带有元数据的质谱
**用例：** 遗留蛋白质组数据
**Python 库：**
- `pyteomics.mzxml`
- `pymzml`：可以读取 mzXML
**EDA 方法：**
- 类似于mzML
- 格式版本兼容性
- 转换质量验证
- 元数据保存检查
