# FluidSim 求解器

FluidSim 为不同的流体动力学方程提供了多个求解器。所有求解器都使用伪谱方法和 FFT 在周期域上工作。

## 可用的求解器

### 二维不可压缩纳维-斯托克斯

**解算器键**：`ns2d`

**导入**：
```python
from fluidsim.solvers.ns2d.solver import Simul
# or dynamically
Simul = fluidsim.import_simul_class_from_key("ns2d")
```

**用途**：二维湍流研究、涡流动力学、基本流体流动模拟

**主要特征**：能量和熵级联、涡度动力学

## 求解器选择指南

根据物理问题选择求解器：

1. **二维湍流，快速测试**：使用`ns2d`
2. **3D 流程，真实模拟**：使用 `ns3d`
3. **密度分层流**：使用 `ns2d.strat` 或 `ns3d.strat`
4. **地球物理流、旋转系统**：使用`sw1l`
5. **弹性板**：使用`fvk`
