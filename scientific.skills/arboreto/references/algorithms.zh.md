# GRN 推理算法

Arboreto 提供了两种基因调控网络 (GRN) 推理算法，均基于多元回归方法。

## 算法概述

两种算法都遵循相同的推理策略：
1. 对于数据集中的每个目标基因，训练回归模型
2. 确定模型中最重要的特征（潜在的监管者）
3. 将这些特征作为具有重要性分数的候选监管者发出

主要区别在于**计算效率**和底层回归方法。

## GRNBoost2（推荐）

**目的**：使用梯度提升对大规模数据集进行快速 GRN 推理。

### 何时使用
- **大型数据集**：数以万计的观察结果（例如，单细胞 RNA-seq）
- **时间受限的分析**：需要比 GENIE3 更快的结果
- **默认选择**：GRNBoost2 是旗舰算法，推荐用于大多数用例

### 技术细节
- **方法**：具有提前停止正则化的随机梯度提升
- **性能**：在大型数据集上比 GENIE3 快得多
- **输出**：与 GENIE3 格式相同（TF-目标-重要性三元组）

### 用法
```python
from arboreto.algo import grnboost2

network = grnboost2(
    expression_data=expression_matrix,
    tf_names=tf_names,
    seed=42  # For reproducibility
)
```

## 选择正确的算法

**决策指南**：

1. **从GRNBoost2开始** - 它速度更快，可以更好地处理大型数据集
2. **使用 GENIE3 如果**：
   - 与现有 GENIE3 出版物的比较
   - 数据集是中小型的
   - 验证 GRNBoost2 结果

两种算法都产生具有相同输出格式的可比较的调节网络，使得它们对于大多数分析来说是可互换的。
