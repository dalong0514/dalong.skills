# 科学写作

## 概述

科学写作是精确、清晰地传达研究成果的过程。使用 IMRAD 结构、引文（APA/AMA/温哥华）、图表/表格和报告指南（CONSORT/STROBE/PRISMA）撰写手稿。将此技能应用于研究论文和期刊提交。

## 何时使用此技能

该技能应该在以下情况下使用：
- 撰写或修改科学手稿的任何部分（摘要、引言、方法、结果、讨论）
- 使用 IMRAD 或其他标准格式构建研究论文
- 以特定样式格式化引文和参考文献（APA、AMA、温哥华、芝加哥、IEEE）
- 创建、格式化或改进图形、表格和数据可视化
- 应用特定于研究的报告指南（试验用 CONSORT、观察性研究用 STROBE、综述用 PRISMA）
- 起草符合期刊要求的摘要（结构化或非结构化）
- 准备稿件以提交给特定期刊
- 提高写作清晰度、简洁性和准确性
- 确保正确使用特定领域的术语和术语
- 处理审稿意见并修改稿件

## 核心能力

### 1. 稿件结构和组织

**IMRAD 格式**：通过大多数科学学科使用的标准引言、方法、结果和讨论结构来指导论文。这包括：
- **简介**：建立研究背景，找出差距，陈述目标
- **方法**：详细研究设计、人群、程序和分析方法
- **结果**：客观地呈现研究结果，无需解释
- **讨论**：解释结果，承认局限性，提出未来方向

有关 IMRAD 结构的详细指南，请参阅`references/imrad_structure.md`。

**替代结构**：支持特定于学科的格式，包括：
- 评论文章（叙述性、系统性、范围界定）
- 病例报告和病例系列
- 荟萃分析和汇总分析
- 理论/模型论文
- 方法论文和协议

### 2.特定部分的写作指导

**摘要写作**：撰写简洁、独立的摘要（100-250 个字），以体现论文的目的、方法、结果和结论。支持结构化摘要（带标记部分）和非结构化单段落格式。

**介绍开发**：构建引人注目的介绍：
- 确定研究问题的重要性
- 系统回顾相关文献
- 识别知识差距或争议
- 陈述明确的研究问题或假设
- 解释该研究的新颖性和意义

**方法文档**：通过以下方式确保可重复性：
- 详细的参与者/样本描述
- 清晰的程序文件
- 有依据的统计方法
- 设备和材料规范
- 道德批准和同意声明

**结果展示**：展示研究结果：
- 从主要结果到次要结果的逻辑流程
- 与图形和表格的集成
- 效应大小的统计显着性
- 客观报告，无需解释

**讨论构建**：通过以下方式综合发现：
- 将结果与研究问题联系起来
- 与现有文献的比较
- 诚实地承认局限性
- 提出机制解释
- 提出实际意义和未来研究

### 3. 引文和参考文献管理

跨学科正确应用引文风格。有关全面的样式指南，请参阅`references/citation_styles.md`。

**主要引文样式：**
- **AMA（美国医学会）**：编号上标引文，在医学中常见
- **温哥华**：方括号内的引用编号，生物医学标准
- **APA（美国心理学会）**：作者日期文本引用，在社会科学中常见
- **芝加哥**：注释-参考书目或作者日期，人文和科学
- **IEEE**：编号方括号，工程和计算机科学

**最佳实践：**
- 尽可能引用主要来源
- 包括最近的文献（活跃领域的最近 5-10 年）
- 平衡介绍和讨论中的引文分布
- 根据原始来源验证所有引用
- 使用参考文献管理软件（Zotero、Mendeley、EndNote）

### 4. 图表

创建有效的数据可视化以增强理解。有关详细的最佳实践，请参阅`references/figures_tables.md`。

**何时使用表格与图表：**
- **表格**：精确的数值数据、复杂的数据集、需要精确值的多个变量
- **数字**：趋势、模式、关系、比较最直观地理解

**设计原则：**
- 使每个表格/图不言自明，并附有完整的标题
- 在所有显示项目中使用一致的格式和术语
- 用单位标记所有轴、列和行
- 包括样本大小 (n) 和统计注释
- 遵循“每 1000 字一张表格/图”的指导原则
- 避免文本、表格和图形之间重复信息

**常见图形类型：**
- 条形图：比较离散类别
- 折线图：显示随时间变化的趋势
- 散点图：显示相关性
- 箱形图：显示分布和异常值
- 热图：可视化矩阵和模式

### 5. 按研究类型划分的报告指南

遵循既定的报告标准，确保完整性和透明度。有关全面的指南详细信息，请参阅`references/reporting_guidelines.md`。

**关键准则：**
- **CONSORT**：随机对照试验
- **STROBE**：观察性研究（队列、病例对照、横断面）
- **PRISMA**：系统评价和荟萃分析
- **STARD**：诊断准确性研究
- **TRIPOD**：预测模型研究
- **到达**：动物研究
- **护理**：病例报告
- **SQUIRE**：质量改进研究
- **SPIRIT**：临床试验研究方案
- **干杯**：经济评估

每项指南都提供了清单，确保报告所有关键方法要素。

### 6.写作原则和风格

应用基本的科学写作原则。有关详细指南，请参阅`references/writing_principles.md`。

**清晰度**：
- 使用精确、明确的语言
- 首次使用时定义技术术语和缩写
- 保持段落内和段落之间的逻辑流程
- 为了清晰起见，在适当的时候使用主动语态

**简洁**：
- 消除多余的单词和短语
- 喜欢较短的句子（平均 15-20 个单词）
- 删除不必要的限定符
- 严格遵守字数限制

**准确度**：
- 以适当的精度报告准确的值
- 始终使用一致的术语
- 区分观察和解释
- 适当地承认不确定性

**客观性**：
- 不带偏见地呈现结果
- 避免夸大调查结果或影响
- 承认相互矛盾的证据
- 保持专业、中立的语气

### 7. 期刊特定格式

使稿件适应期刊要求：
- 遵循作者的结构、长度和格式指南
- 应用期刊特定的引文风格
- 满足图形/表格规格（分辨率、文件格式、尺寸）
- 包括所需的声明（资金、利益冲突、数据可用性、道德批准）
- 遵守每个部分的字数限制
- 根据提供的模板要求进行格式设置

### 8. 特定领域的语言和术语

调整语言、术语和惯例以匹配特定的科学学科。每个领域都建立了词汇表、首选措辞和特定领域的惯例，以表明专业知识并确保目标受众的清晰度。

**确定特定领域的语言约定：**
- 审查目标期刊近期高影响力论文中使用的术语
- 注意特定领域的缩写、单位和符号系统
- 确定首选术语（例如，“参与者”与“受试者”、“化合物”与“药物”、“样本”与“样品”）
- 观察通常如何描述方法、有机体或技术

**生物医学和临床科学：**
- 使用精确的解剖学和临床术语（例如，正式写作中使用“心肌梗塞”而不是“心脏病发作”）
- 遵循标准化疾病命名法（ICD、DSM、SNOMED-CT）
- 首先使用通用名称指定药物名称，如果需要，在括号中指定品牌名称
- 使用“患者”进行临床研究，使用“参与者”进行社区研究
- 遵循人类基因组变异学会 (HGVS) 的遗传变异命名法
- 使用标准单位报告实验室值（大多数国际期刊中的 SI 单位）

**分子生物学和遗传学：**
- 基因符号使用斜体（例如，*TP53*），蛋白质使用常规字体（例如，p53）
- 遵循物种特异性基因命名法（人类大写：*BRCA1*；小鼠句子大小写：*Brca1*）
- 在第一次提及时指定完整的生物体名称，然后使用可接受的缩写（例如，*大肠杆菌*，然后是*大肠杆菌*）
- 使用标准遗传符号（例如，+/+、+/-、-/- 表示基因型）
- 采用分子技术的既定术语（例如“定量 PCR”或“qPCR”，而不是“实时 PCR”）

**化学和制药科学：**
- 遵循 IUPAC 化合物命名法
- 对新化合物使用系统名称，对众所周知的物质使用通用名称
- 使用标准符号指定化学结构（例如数据库的 SMILES、InChI）
- 使用适当的单位报告浓度（mM、μM、nM 或 % w/v、v/v）
- 使用公认的反应命名法描述合成路线
- 使用与领域定义一致的术语，如“生物利用度”、“药代动力学”、“IC50”

**生态与环境科学：**
- 对物种使用二项式命名法（斜体：*智人*）
- 在相关时在第一个物种提及时指定分类当局
- 采用标准化栖息地和生态系统分类
- 对生态指标使用一致的术语（例如“物种丰富度”、“香农多样性指数”）
- 使用现场标准术语描述采样方法（例如“样线”、“样方”、“标记重新捕获”）

**物理与工程：**
- 一致遵循 SI 单位，除非现场惯例另有规定
- 使用物理量的标准表示法（标量与向量、张量）
- 采用现象的既定术语（例如“量子纠缠”、“层流”）
- 指定相关设备的型号和制造商
- 使用符合现场标准的数学符号（例如，ℏ表示简化的普朗克常数）

**神经科学：**
- 使用标准化的大脑区域命名法（例如，参考艾伦脑图谱等地图集）
- 使用已建立的立体定位系统指定大脑区域的坐标
- 遵循神经术语惯例（例如，正式写作中的“动作电位”而不是“尖峰”）
- 根据测量方法适当使用“神经活动”、“神经元放电”、“大脑激活”
- 描述具有适当特异性的记录技术（例如，“全细胞膜片钳”、“细胞外记录”）

**社会和行为科学：**
- 适当时使用以人为本的语言（例如，“精神分裂症患者”而不是“精神分裂症患者”）
- 采用标准化的心理结构和经过验证的评估名称
- 遵循 APA 指南以减少语言偏见
- 使用既定术语指定理论框架
- 使用“参与者”而不是“受试者”进行人类研究

**一般原则：**

**匹配观众专业知识：**
- 对于专业期刊：自由使用特定领域术语，仅定义高度专业化或新颖的术语
- 对于具有广泛影响力的期刊（例如，*Nature*、*Science*）：定义更多技术术语，为专业概念提供上下文
- 对于跨学科受众：平衡精确性与可访问性，在首次使用时定义术语

**战略性地定义技术术语：**
- 首次使用时定义缩写：“信使 RNA (mRNA)”
- 为更广泛的受众写作时提供专业技巧的简要解释
- 避免过度定义目标受众熟知的术语（表明不熟悉该领域）
- 如果大量专业术语不可避免，则创建一个术语表

**保持一致性：**
- 始终使用相同的术语来表示相同的概念（不要在“药物”、“药物”和“药物”之间交替使用）
- 遵循一致的缩写系统（在第一次定义后决定“PCR”或“聚合酶链式反应”）
- 始终应用相同的命名系统（特别是基因、物种、化学物质）

**避免现场混合错误：**
- 不要使用基础科学的临床术语（例如，不要称小鼠为“患者”）
- 避免使用口语或过于笼统的术语来代替精确的领域术语
- 在未确保正确使用的情况下，不要从相邻领域导入术语

**验证术语用法：**
- 查阅特定领域的风格指南和术语资源
- 检查目标期刊最近论文中如何使用术语
- 使用特定领域的数据库和本体（例如基因本体、MeSH 术语）
- 当不确定时，引用建立术语的关键参考文献

### 9. 要避免的常见陷阱

**主要拒绝原因：**
1. 统计数据描述不恰当、不完整或不充分
2. 对结果的过度解释或无支持的结论
3. 方法描述不当影响重现性
4. 样本量小、有偏见或不适当
5. 写作质量差或文字难以理解
6. 文献综述或背景不充分
7、图表不清晰或设计不当
8. 不遵守报告准则

**写作质量问题：**
- 时态混合不当（方法/结果使用过去时，既定事实使用现在时）
- 过多的行话或未定义的首字母缩略词
- 段落中断会破坏逻辑流程
- 部分之间缺少过渡
- 符号或术语不一致

## 稿件开发工作流程

**第一阶段：规划**
1. 确定目标期刊和审稿作者指南
2. 确定适用的报告指南（CONSORT、STROBE 等）
3. 概要稿件结构（通常是IMRAD）
4. 规划图表作为论文的支柱

**第二阶段：起草**
1. 从数字和表格开始（核心数据故事）
2. 编写方法（通常最容易先起草）
3. 结果草案（客观描述数字/表格）
4. 撰写讨论（解释调查结果）
5.写引言（设置研究问题）
6. Craft Abstract（综合完整的故事）
7. 创建标题（简洁且具有描述性）

**第三阶段：修订**
1. 检查整个逻辑流程和“红线”
2. 验证术语和符号的一致性
3. 确保图表不言自明
4. 确认遵守报告准则
5. 验证所有引文是否准确且格式正确
6. 检查每个部分的字数
7. 校对语法、拼写和清晰度

**第四阶段：最后准备**
1.格式按期刊要求
2.准备补充材料
3.写求职信强调重要性
4. 填写提交清单
5. 收集所有必需的报表和表格
