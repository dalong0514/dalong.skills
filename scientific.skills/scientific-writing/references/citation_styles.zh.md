# 引文风格指南

## 概述

引文样式提供了科学写作中确认来源的标准化格式。不同的学科喜欢不同的风格，期刊通常会指定使用哪种风格。科学领域最常见的五种引文风格是 AMA、Vancouver、APA、Chicago 和 IEEE。

## 选择正确的风格

|风格|主要学科|文本格式|
|------|--------------------------------|----------------|
|阿玛 |医学、健康科学|上标数字1 |
|温哥华 |生物医学|括号中的数字 [1] |
| APA |心理学、社会科学、教育|作者日期（史密斯，2023 年）|
|芝加哥 |人文、历史、一些科学|注释-参考书目或作者日期|
| IEEE |工程、计算机科学|括号中的数字 [1] |
| ACS |化学|上标数字 1 或 (1) |
|国家LM |生命科学、PubMed |括号中的数字 [1] |

**默认推荐**：如有疑问，请查看期刊的作者指南。大多数生物医学期刊使用温哥华或 AMA 风格。

## AMA 风格（美国医学会）

### 概述
- 主要用于医学研究
- 基于*AMA 风格手册*（2020 年第 11 版）
- 编号的引文显示为上标
- 参考文献按出现顺序按数字列出

### 文本引用

**基本格式**：上标数字位于句点和逗号之外，分号和冒号内部。

**示例：**
```
Several studies have demonstrated this effect.¹

The results were inconclusive,² although Smith et al³ reported otherwise.

These findings³⁻⁵ suggest a correlation.

One meta-analysis⁶ found significant heterogeneity; however, the pooled effect was significant.⁷
```

**多重引用**：使用逗号或连字符表示范围
<<<代码块_1>>>

**多次引用同一来源**：自始至终使用相同的数字

### 参考列表格式

**期刊文章：**
<<<代码块_2>>>

**示例：**
<<<代码块_3>>>

**书籍：**
<<<代码块_4>>>

**书籍章节：**
<<<代码块_5>>>

**在线资源：**
<<<代码块_6>>>

### 特殊情况

**超过 6 位作者**：先列出 3 位，然后列出“等人”
```
Smith JD, Jones AB, Williams CD, et al.
```

**无作者**：以标题开头

**提前在线发布**：
```
Published online Month Day, Year. doi:xx.xxxx
```

## 温哥华风格

### 概述
- 由国际医学期刊编辑委员会 (ICMJE) 制定
- 在*关于医学期刊学术工作的进行、报告、编辑和出版的建议*中进行了描述
- 也称为“作者编号样式”
- 方括号内的编号引文
- 按数字列出的参考文献

### 文本引用

**基本格式**：相关文本后面、句号和逗号前面的方括号中的数字。

**示例：**
```
Several studies have shown this effect [1].

The results were inconclusive [2], although Smith et al [3] reported otherwise.

These findings [3-5] suggest a correlation.

Multiple studies [1,3,5-7] have confirmed this.
```

### 参考列表格式

**期刊文章：**
```
1. Author AA, Author BB, Author CC. Title of article. Journal Name. Year;Volume(Issue):Page range.
```

**示例：**
```
1. Smith JD, Johnson AB, Williams CD. Effectiveness of cognitive behavioral therapy for anxiety disorders. JAMA Psychiatry. 2023;80(5):456-464.
```

**书籍：**
```
2. Author AA, Author BB. Book title. Edition. Place of publication: Publisher; Year.
```

**书籍章节：**
```
3. Chapter Author AA, Chapter Author BB. Chapter title. In: Editor AA, Editor BB, editors. Book title. Edition. Place: Publisher; Year. p. Page range.
```

**电子资源：**
```
4. Author AA. Title of page [Internet]. Place: Publisher; Date of publication [cited Date of citation]. Available from: URL
```

### 特殊情况

**超过 6 位作者**：先列出 6 位作者，然后列出“等人”。

**期刊标题缩写**：使用 PubMed/Index Medicus 缩写
- *美国医学会杂志* → *JAMA*
- *自然医学* → *自然医学*

**无卷或期**：仅使用年份和页码

**印刷中的文章**：使用“[Epub before print]”符号

## APA风格（美国心理学会）

### 概述
- 广泛应用于心理学、教育学和社会科学
- 基于 *APA 出版手册*（2020 年第 7 版）
- 文本引用的作者日期格式
- 参考文献按作者姓氏字母顺序排列

### 文本引用

**基本格式**：（作者、年份）

**示例：**
```
One study found significant effects (Smith, 2023).

Smith (2023) found significant effects.

Multiple studies (Jones, 2020; Smith, 2023; Williams, 2024) support this conclusion.
```

**两位作者**：括号中使用“&”，叙述中使用“and”
```
(Smith & Jones, 2023)
Smith and Jones (2023) demonstrated...
```

**三位或更多作者**：使用“等人”。在第一作者之后
```
(Smith et al., 2023)
Smith et al. (2023) reported...
```

**同一作者同一年的多部作品**：添加字母
```
(Smith, 2023a, 2023b)
```

**直接引用**：包括页码
```
(Smith, 2023, p. 45)
"Quote text" (Smith, 2023, p. 45).
Smith (2023) stated, "Quote text" (p. 45).
```

### 参考列表格式

**期刊文章：**
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Journal Name, Volume(Issue), page range. https://doi.org/xx.xxxx
```

**示例：**
```
Smith, J. D., Johnson, A. B., & Williams, C. D. (2023). Effectiveness of cognitive behavioral therapy for anxiety disorders. JAMA Psychiatry, 80(5), 456-464. https://doi.org/10.1001/jamapsychiatry.2023.0123
```

**书籍：**
```
Author, A. A. (Year). Book title: Subtitle (Edition). Publisher. https://doi.org/xx.xxxx
```

**书籍章节：**
```
Chapter Author, A. A., & Chapter Author, B. B. (Year). Chapter title. In E. E. Editor & F. F. Editor (Eds.), Book title (pp. page range). Publisher.
```

**网站：**
```
Author, A. A. (Year, Month Day). Page title. Website Name. URL
```

### 大写规则
- 文章和书名的句首字母大写（仅第一个单词和专有名词大写）
- 期刊名称的标题大小写（所有主要单词大写）

**示例：**
```
Smith, J. D. (2023). Effects of stress on cognitive performance: A meta-analysis. Journal of Experimental Psychology: General, 152(3), 456-478.
```

### 特殊情况

**无作者**：将标题移至作者位置
```
Title of work. (Year). Journal Name...
```

**无日期**：使用（n.d.）
```
Smith, J. D. (n.d.). Title...
```

**最多 20 位作者**：列出最后一个前面带有“&”的所有作者
**21 位或更多作者**：先列出 19 位作者，然后列出“...”，然后是最终作者

## 芝加哥风格

### 概述
- 基于*《芝加哥风格手册》*（2017 年第 17 版）
- 两个系统：注释-参考书目和作者-日期
- 注释-人文学科中常见的参考书目
- 科学中常见的作者日期

### 注释-参考书目系统

**文本内**：脚注或尾注的上标数字
```
One study demonstrated this effect.¹
```

**注释格式：**
```
1. John D. Smith, Alice B. Johnson, and Carol D. Williams, "Effectiveness of Cognitive Behavioral Therapy for Anxiety Disorders," JAMA Psychiatry 80, no. 5 (2023): 456-64.
```

**参考书目格式：**
```
Smith, John D., Alice B. Johnson, and Carol D. Williams. "Effectiveness of Cognitive Behavioral Therapy for Anxiety Disorders." JAMA Psychiatry 80, no. 5 (2023): 456-64.
```

### 作者日期系统

**文本内**：类似于 APA
```
(Smith, Johnson, and Williams 2023)
Smith, Johnson, and Williams (2023) found...
```

**参考列表**：与 APA 类似，但标点符号不同
```
Smith, John D., Alice B. Johnson, and Carol D. Williams. 2023. "Effectiveness of Cognitive Behavioral Therapy for Anxiety Disorders." JAMA Psychiatry 80 (5): 456-64.
```

### 特色
- 参考书目中的全名（不仅仅是缩写）
- 使用“and”而不是“&”
- 与APA不同的标点符号

## IEEE 风格

### 概述
- 用于工程、计算机科学和技术
- 由电气和电子工程师协会出版
- 方括号内的编号引文
- 按数字列出的参考文献

### 文本引用

**格式**：方括号中的数字

**示例：**
```
Several studies have demonstrated this effect [1].

The algorithm was described by Smith [2] and later improved [3], [4].

Multiple implementations [1]-[4] have been proposed.
```

### 参考列表格式

**期刊文章：**
```
[1] A. A. Author, B. B. Author, and C. C. Author, "Title of article," Journal Name, vol. X, no. X, pp. XX-XX, Month Year.
```

**示例：**
```
[1] J. D. Smith, A. B. Johnson, and C. D. Williams, "Effectiveness of cognitive behavioral therapy for anxiety disorders," JAMA Psychiatry, vol. 80, no. 5, pp. 456-464, May 2023.
```

**书籍：**
```
[2] A. A. Author, Book Title, Edition. City, State: Publisher, Year.
```

**会议论文：**
```
[3] A. A. Author, "Paper title," in Proc. Conference Name, City, State, Year, pp. XX-XX.
```

**在线资源：**
```
[4] A. A. Author. "Title." Website. URL (accessed Mon. Day, Year).
```

### 特色
- 名字和中间名的缩写
- 在最后一位作者之前使用“and”（不是逗号）
- 月份缩写（Jan.、Feb. 等）
- “卷”和“不”。在卷和发行之前
- “页”页面范围之前

## 其他样式

### ACS 风格（美国化学会）

**文本内**：上标数字或括号中的数字
```
This reaction has been well studied.¹
This reaction has been well studied (1).
```

**参考格式：**
```
(1) Smith, J. D.; Johnson, A. B.; Williams, C. D. Title of Article. J. Am. Chem. Soc. 2023, 145, 1234-1245.
```

**特点：**
- 作者之间的分号
- 期刊名称缩写
- 粗体年份
- 没有发行号

### NLM 风格（国家医学图书馆）

**与温哥华非常相似**，由 PubMed/MEDLINE 使用

**主要区别：**
- 使用 PubMed 期刊缩写
- 电子出版物的特定格式
- 可以包括 PMID 或 PMCID

**示例：**
```
Smith JD, Johnson AB, Williams CD. Effectiveness of cognitive behavioral therapy for anxiety disorders. JAMA Psychiatry. 2023 May;80(5):456-64. doi: 10.1001/jamapsychiatry.2023.0123. PMID: 12345678.
```

## 一般引文最佳实践

### 跨越所有风格

**何时引用：**
- 直接报价
- 转述他人的想法
- 来自其他来源的统计数据、数据或数字
- 其他人开发的理论、模型或框架
- 不属于常识的信息

**引用密度：**
- 简介：自由引用以建立上下文
- 方法：在引用既定方案或文书时引用
- 结果：很少引用（专注于你自己的发现）
- 讨论：与之前的工作进行比较时经常引用

**来源质量：**
- 更喜欢同行评审的期刊文章
- 尽可能引用原始来源（而不是二次引用）
- 使用最新来源（活跃领域 5-10 年内）
- 确保来源信誉良好且相关

**要避免的常见错误：**
- 格式不一致
- 缺少必需的元素（DOI、页码等）
- 引用未实际阅读的来源（引用链）
- 过度依赖评论文章而不是主要来源
- 包括未引用的参考文献或缺失的引用参考文献
- 作者姓名或缩写不正确
- 出版年份错误
- 标题被截断

### 管理引文

**参考管理软件：**
- **Zotero**：免费、开源、浏览器集成
- **Mendeley**：免费、PDF 注释、社交功能
- **尾注**：商业、强大、机构支持
- **RefWorks**：基于网络的机构订阅

**软件优势：**
- 多种样式自动格式化
- 文本内引用插入
- 参考列表生成
- PDF组织
- 共享能力

### 验证引文

**提交前，请检查：**
1. 每个文内引文都有相应的参考文献
2. 每条参考文献均在正文中引用
3. 格式自始至终保持一致
4.作者姓名及姓名首字母正确
5. 标题准确
6. 期刊名称与所需缩写相符
7. 卷数、期数、页码正确
8. 包含 DOI（需要时）
9. URL 具有功能性（对于网络资源）
10. 引文以正确的顺序出现（数字样式）

## DOI（数字对象标识符）

### 什么是 DOI？
永久标识数字内容的唯一字母数字字符串。

**格式：**
```
doi:10.1001/jamapsychiatry.2023.0123
or
https://doi.org/10.1001/jamapsychiatry.2023.0123
```

### 何时包含：
- 大多数期刊要求最近发表的文章
- 优先于 URL，因为 DOI 不会改变
- 如果未提供，请在 https://www.crossref.org/ 查找 DOI

### 特定于样式的格式：
- **AMA**：`doi:10.xxxx/xxxxx`
- **APA**：`https://doi.org/10.xxxx/xxxxx`
- **温哥华**：通常由期刊酌情省略或添加
- **芝加哥**：`https://doi.org/10.xxxx/xxxxx`

## 常用缩写

### 期刊缩写
遵循期刊的指定系统（通常是 Index Medicus 或 ISO）：
- *生物化学杂志* → *J Biol Chem*
- *美国国家科学院院刊* → *Proc Natl Acad Sci USA*
- *自然医学* → *自然医学*

### 月份缩写
- 一月、二月、三月、四月、五月、六月、七月、八月、九月、十月、十一月、十二月
- 某些样式使用不带句点的三字母缩写

### 版本缩写
- 第一版、第二版、第三版等
- 或：第一版、第二版

## 特别出版物类型

### 预印本


### 论文和论文
```

```

### 会议记录
```

```

```

```

```

```

## 风格之间的转换

在引用样式之间转换时：

1. **使用参考文献管理软件**进行自动转换
2. **检查这些因风格而异的元素**：
   - 文本引用格式（编号与作者日期）
   - 作者姓名格式（缩写与全名）
   - 标题大写（句子大小写与标题大小写）
   - 期刊名称格式（缩写与完整）
   - 标点符号（句号、逗号、分号）
   - 使用斜体和粗体
   - 元素的顺序
3. **自动转换后手动验证**
4. **查看期刊指南**了解具体要求

## 期刊特定的引文样式和要求

### 如何识别期刊的引文风格

**第 1 步：检查作者指南**
- 每个期刊都提供作者说明（通常是“作者说明”或“作者指南”）
- 引文样式通常在“参考文献”或“引文”部分指定
- 查找以期刊风格格式化的参考文献示例

**第 2 步：查看最近发表的文章**
- 检查目标期刊最近的 3-5 篇文章
- 注意文本引用格式（编号与作者日期）
- 比较参考列表格式
- 检查期刊特定的变化

**步骤 3：验证期刊特定的变体**
一些期刊使用标准样式的修改版本：
- 期刊名称缩写与完整名称
- DOI 包含要求
- 标题大小写与句子大小写的文章标题
- “et al.”之前的作者最大数量

| **科学** |科学风格（编号）|文本内编号，缩写格式|

### 期刊系列一致性

**来自同一出版商的期刊通常共享引文风格：**

**爱思唯尔期刊：**
- 差异很大；检查具体期刊
- 许多使用温哥华风格的编号
- 有些允许作者日期

**施普林格自然期刊：**
- 自然期刊：自然风格（编号、缩写）
- Springer 期刊：通常根据领域进行编号或作者日期
- BMC 期刊：温哥华期刊全名

**Wiley 期刊：**
- 因领域而异
- 许多生物医学期刊使用温哥华
- 心理学/社会科学期刊经常使用APA

**美国化学会 (ACS)：**
- 所有 ACS 期刊均使用 ACS 风格
- 与《美国化学会杂志》、《分析化学》等杂志一致。

### 调整不同目标期刊的引文

**在桌面拒绝或撤回后切换期刊时：**

**使用参考文献管理软件：**
1. 将参考文献导入 Zotero、Mendeley 或 EndNote
2. 从软件库中选择目标期刊的引文风格
3.自动重新生成引文和参考文献列表
4. 手动验证格式是否与日志示例匹配

**转换时要检查的关键要素：**
- 文本内格式（开关编号↔作者日期）
- 期刊名称缩写样式
- 文章标题大写
- 作者姓名格式（缩写与全名）
- DOI 格式和包含
- 问题编号包含/排除
- 页码格式

**手动验证对于以下情况至关重要：**
- 预印本和非标准来源
- 软件/数据集引用
- 会议记录
- 论文和论文

### 场地特定评估标准

**内容期望：**
- **高影响力期刊**：过去 5 年的引用率 >50%；首选主要来源
- **医学期刊**：最近的临床证据；系统评价的价值
- **机器学习会议**：最近的论文（过去 2-3 年）；预印本 (arXiv) 可接受
- **自引**：在所有场所保持 <20%

**格式合规性（通常是自动化的）：**
- 完全匹配场地引用风格
- 所有文内引文都有相应的参考文献
- 需要时包含 DOI（期刊）或 arXiv ID（机器学习会议）
- 使用正确的缩写（PubMed 用于医学，标准用于 ML）

**机器学习会议细节：**
- **NeurIPS/ICML/ICLR**：ArXiv 预印本被广泛引用；最近的工作受到高度重视
- **页面限制严格**：引文格式影响空间
- **补充材料**：可以包括扩展参考书目
- **双盲评审**：评审过程中避免明显的自引模式

### 按地点类型划分的引用密度

|场地类型 |预期引用次数 |要点|
|----------|--------------------|------------|
| **自然/科学研究** | 30-50 | 30-50选择性、高影响力的引用 |
| **医学期刊（RCT）** | 25-40 | 25-40最近的临床证据|
| **特定领域期刊** | 30-60 | 30-60全面的领域覆盖 |
| **机器学习会议（8 页）** | 20-40 | 20-40篇幅有限，近期作品|
| **评论文章** | 100-300+ |全面覆盖|

**机器学习会议引用实践：**
- **NeurIPS/ICML**：8 页论文的典型参考文献为 25-40 篇
- **研讨会论文**：15-25 篇参考文献
- **ArXiv 预印本**：被广泛接受和引用
- **相关工作**：简洁但全面；经常移至附录
- **近期关键**：引用过去 1-2 年的相关工作

### 提交前引文清单

**内容：**
- [ ] 过去 5-10 年（或 ML 会议 2-3 年）的引用率≥50%
- [ ] <20% 自引；平衡的观点
- [ ] 引用的主要来源（不是引用链）
- [ ] 所有主张均由适当的引文支持

**格式：**
- [ ] 风格与场地完全匹配（检查模板）
- [ ] 参考列表中的所有文本引用，反之亦然
- [ ] 根据需要包含 DOI/arXiv ID
- [ ] 缩写匹配场地风格

**机器学习会议附加：**
- [ ] ArXiv 预印本格式正确
- [ ] 如果双盲评审，自引将匿名
- [ ] 参考文献不超过页数限制
