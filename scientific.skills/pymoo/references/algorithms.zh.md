# Pymoo 算法参考

pymoo 中提供的优化算法的综合参考。

## 单目标优化算法

### 遗传算法 (GA)
**用途：**通用单目标进化优化
**最适合：**连续、离散或混合变量问题
**算法类型：** (μ+λ)遗传算法

**关键参数：**
- `pop_size`：人口规模（默认值：100）
- `sampling`：初始种群生成策略
- `selection`：父级选择机制（默认：锦标赛）
- `crossover`：重组运算符（默认值：SBX）
- `mutation`：变分运算符（默认值：多项式）
- `eliminate_duplicates`：删除冗余解决方案（默认值：True）
- `n_offsprings`：每代的后代

**用途：**
```python
from pymoo.algorithms.soo.nonconvex.ga import GA
algorithm = GA(pop_size=100, eliminate_duplicates=True)
```

### 差分进化（DE）
**目的：** 单目标持续优化
**最适合：** 通过良好的全局搜索进行连续参数优化
**算法类型：** 基于群体的差分进化

**变体：** 提供多种 DE 策略（rand/1/bin、best/1/bin 等）

### 粒子群优化 (PSO)
**目的：**通过群体智能进行单目标优化
**最适合：** 连续问题，平滑景观上的快速收敛

### CMA-ES
**目的：**协方差矩阵适应进化策略
**最适合：** 持续优化，特别是对于噪声或病态问题

### 模式搜索
**用途：**直接搜索方法
**最适合：** 梯度信息不可用的问题

### 内尔德-米德
**目的：** 基于单纯形的优化
**最适合：** 连续函数的局部优化

## 算法选择指南

**对于单目标问题：**
- 从GA开始解决一般问题
- 使用DE进行持续优化
- 尝试 PSO 来更快地解决平滑问题
- 对于困难/嘈杂的景观使用 CMA-ES

**对于多目标问题：**
- 2-3 个目标：NSGA-II
- 4 个以上目标：NSGA-III
首选关节：R-NSGA-II
- 易于分解：MOEA/D
- 超容量焦点：SMS-EMOA

**对于受限问题：**
- 基于可行性的生存选择（适用于大多数算法）
- 严格限制：SRES/ISRES
- 算法兼容性的惩罚方法

**对于动态问题：**
- 时变：D-NSGA-II
- 有用的历史知识：KGB-DMOEA
