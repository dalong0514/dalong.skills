# 文献数据库检索策略

本文件为系统有效地检索多个文献数据库提供了全面的指导。

## 可用的数据库和技能

### 生物医学与生命科学

#### PubMed / PubMed 中心
- **访问**：使用`gget`技能或WebFetch工具
- **覆盖范围**：生物医学文献中的引用次数超过 3500 万次
- **最适合**：临床研究、生物医学研究、遗传学、分子生物学
- **搜索技巧**：使用 MeSH 术语、布尔运算符（AND、OR、NOT）、字段标签 [标题]、[作者]
- **示例**：`"CRISPR"[Title] AND "gene editing"[Title/Abstract] AND 2020:2024[Publication Date]`

#### bioRxiv / medRxiv
- **访问**：使用`gget`技能或直接API
- **覆盖范围**：生物学和医学预印本
- **最适合**：最新未发表的研究、前沿发现
- **注**：未经同行评审；谨慎核实调查结果
- **搜索技巧**：按类别搜索（生物信息学、基因组学等）

### 一般科学文献

#### arXiv
- **访问**：直接API访问
- **覆盖范围**：物理学、数学、计算机科学、定量生物学的预印本
- **最适合**：计算方法、生物信息学算法、理论工作
- **类别**：q-bio（定量生物学）、cs.LG（机器学习）、stat.ML（统计）
- **搜索格式**：`cat:q-bio.QM AND title:"single cell"`

#### 语义学者
- **访问**：直接 API（需要 API 密钥）
- **覆盖范围**：所有领域超过 2 亿篇论文
- **最适合**：跨学科检索、引文图、论文推荐
- **特色**：有影响力的引文、论文摘要、相关论文
- **速率限制**：使用 API 密钥 100 个请求/5 分钟

#### 谷歌学术
- **访问**：网页抓取（谨慎使用）或手动搜索
- **覆盖范围**：全面覆盖所有领域
- **最适合**：查找高被引用论文、会议记录、论文
- **限制**：没有官方API，速率限制
- **导出**：使用“引用”功能进行格式化引文

### 专业数据库

#### ChEMBL / PubChem
- **访问**：使用`gget`技能或`bioservices`技能
- **覆盖范围**：化合物、生物活性数据、药物分子
- **最适合**：药物发现、化学生物学、药物化学
- **ChEMBL**：2M+ 化合物，生物活性数据
- **PubChem**：110M+ 化合物，分析数据

#### UniProt
- **访问**：使用`gget`技能或`bioservices`技能
- **覆盖范围**：蛋白质序列和功能信息
- **最适合**：蛋白质研究、序列分析、功能注释
- **搜索依据**：蛋白质名称、基因名称、生物体、功能

#### KEGG（京都基因和基因组百科全书）
- **访问**：使用`bioservices`技能
- **覆盖范围**：途径、疾病、药物、基因
- **最适合**：通路分析、系统生物学、代谢研究

#### COSMIC（癌症体细胞突变目录）
- **访问**：使用`gget`技能或直接下载
- **覆盖范围**：癌症基因组学、体细胞突变
- **最适合**：癌症研究、突变分析

#### AlphaFold 数据库
- **访问**：通过 `alphafold` 命令使用 `gget` 技能
- **覆盖率**：200M+ 蛋白质结构预测
- **最适合**：结构生物学、蛋白质建模

#### PDB（蛋白质数据库）
- **访问**：使用`gget`或直接API
- **覆盖范围**：蛋白质、核酸的实验 3D 结构
- **最适合**：结构生物学、药物设计、分子建模

### 引文和参考文献管理

#### 开放亚历克斯
- **访问**：直接 API（免费，无需密钥）
- **覆盖范围**：2.5亿+作品，全面的元数据
- **最适合**：引文分析、作者消歧、机构研究
- **特点**：开放获取，非常适合文献计量学

#### 尺寸
- **访问**：提供免费套餐
- **覆盖范围**：出版物、资助、专利、临床试验
- **最适合**：研究影响、资金分析、转化研究

## 搜索策略框架

### 1. 定义研究问题（PICO 框架）

对于临床/生物医学审查：
- **P**opulation：研究对象是谁？
- **I**干预：正在测试什么？
- **C**比较：与什么相比？
- **O**结果：结果是什么？

**示例**：“与标准治疗 (C) 相比，CRISPR-Cas9 基因疗法 (I) 治疗镰状细胞病 (P) 在改善患者预后 (O) 方面的功效如何？”

### 2. 开发搜索词

#### 主要概念
从您的研究问题中找出 2-4 个主要概念。
**示例**：
- 概念1：CRISPR、Cas9、基因编辑
- 概念 2：镰状细胞病、SCD、血红蛋白疾病
- 概念3：基因治疗、治疗性编辑

#### 同义词和相关术语
列出替代术语、缩写和相关概念。

**工具**：使用 MeSH（医学主题词）浏览器获取标准化术语

#### 布尔运算符
- **AND**：缩小搜索范围（必须包含两个术语）
- **或**：扩大搜索范围（包括任一术语）
- **NOT**：排除条款

**示例**：`(CRISPR OR Cas9 OR "gene editing") AND ("sickle cell" OR SCD) AND therapy`

#### 通配符和截断
- `*` 或 `%`：匹配任何字符
- `?`：匹配单个字符

**示例**：`genom*` 匹配基因组、基因组、基因组

### 3. 设置包含/排除标准

#### 纳入标准
- **日期范围**：例如，2015-2024（过去 10 年）
- **语言**：英语（或指定多语言）
- **出版物类型**：同行评审的文章、评论、预印本
- **研究设计**：随机对照试验、队列研究、荟萃分析
- **人群**：人类、动物模型、体外

#### 排除标准
- 病例报告（n<5）
- 无全文的会议摘要
- 非原创研究（社论、评论）
- 重复出版物
- 撤回文章

### 4.数据库选择策略

#### 多数据库方法
搜索至少 3 个互补数据库：

1. **主要数据库**：PubMed（生物医学）或 arXiv（计算）
2. **预印本服务器**：bioRxiv/medRxiv 或 arXiv
3. **综合数据库**：Semantic Scholar 或 Google Scholar
4. **专业数据库**：ChEMBL、UniProt 或特定领域

#### 数据库特定语法

|数据库|字段标签 |示例|
|----------|------------|---------|
|考研| [标题]、[作者]、[MeSH] | “CRISPR”[标题] 和 2020:2024[DP] |
| arXiv | ti:, au:, cat: | ti：“机器学习”和cat：q-bio.QM |
|语义学者|标题：，作者：，年份： |标题：《深度学习》 年份：2020-2024 |

## 搜索执行工作流程

### 第一阶段：飞行员搜索
1. 使用广义术语进行初步搜索
2. 查看前 50 个结果的相关性
3. 注意常见关键词和 MeSH 术语
4. 细化搜索策略

### 第二阶段：综合搜索
1. 在所有选定的数据库中执行精细搜索
2. 以标准格式导出结果（RIS、BibTeX、JSON）
3. 每个数据库的文档搜索字符串和日期
4. 记录每个数据库的结果数

### 第 3 阶段：重复数据删除
1. 将所有结果导入到一个文件中
2.使用`search_databases.py --deduplicate`删除重复项
3. 通过 DOI（主要）或标题（后备）识别重复项
4.保留元数据最完整的版本

### 第四阶段：筛选
1. **标题筛选**：审查标题，排除明显不相关的内容
2. **摘要筛选**：阅读摘要，应用纳入/排除标准
3. **全文筛选**：获取并审阅全文
4.记录每个阶段排除的原因

### 第 5 阶段：质量评估
1. 使用适当的工具评估研究质量：
   - **RCT**：Cochrane 偏差风险工具
   - **观察**：纽卡斯尔-渥太华量表
   - **系统评论**：AMSTAR 2
2. 证据质量等级（高、中、低、极低）
3. 考虑排除质量极低的研究

## 高级搜索技术

### 引文链接

#### 正向引文检索
查找引用关键论文的论文：
- 使用谷歌学术搜索“被引用”功能
- 使用 OpenAlex 或 Semantic Scholar API
- 确定基于开创性工作的更新研究

#### 向后引文检索
回顾关键论文中的参考文献：
- 从包含的论文中提取参考文献
- 搜索被引用次数最多的参考文献
- 确定基础研究

### 雪球采样
1. 从 3-5 篇高度相关的论文开始
2. 提取所有参考文献
3.检查哪些参考文献被多篇论文引用
4.审查那些高度重叠的参考文献
5. 对新确定的关键论文重复上述步骤

### 作者搜索
关注该领域多产的作者：
- 在数据库中按作者姓名搜索
- 检查作者简介（ORCID、Google Scholar）
- 回顾最近的出版物和预印本
### 相关文章功能
许多数据库推荐相关文章：
- PubMed“类似文章”
——语义学者《推荐论文》
- 用于发现关键词搜索遗漏的论文

## 要避免的常见陷阱

1. **搜索范围太窄**：缺少相关论文
   - 解决方案：包括同义词、相关术语、更广泛的概念

2. **搜索范围太广**：数千个不相关的结果
   - 解决方案：用AND添加具体概念，使用字段标签

3. **单一数据库**：覆盖不全
   - 解决方案：搜索至少 3 个互补数据库

4. **忽略预印本**：缺少最新发现
   - 解决方案：包括 bioRxiv、medRxiv 或 arXiv

5. **没有文档**：不可重复的搜索
   - 解决方案：记录每个搜索字符串、日期和结果计数

6. **手动重复数据删除**：耗时且容易出错
   - 解决方案：使用search_databases.py脚本

7. **未经验证的引用**：DOI 损坏、元数据不正确
   - 解决方案：在最终参考列表上运行 verify_itations.py

8. **发表偏倚**：仅包括已发表的阳性结果
   - 解决方案：搜索试验注册处，联系作者获取未发表的数据
