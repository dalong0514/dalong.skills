# 实验设计模式

## 检验科学假设的常用方法

该参考文献提供了跨科学领域设计实验的模式和框架。使用这些模式为生成的假设开发严格的测试。

## 设计选型框架

根据以下因素选择实验方法：
- **假设的性质：** 机械性、因果性、相关性、描述性
- **系统研究：** 体外、体内、计算、观察
- **可行性：**时间、成本、道德、技术能力
- **所需证据：** 概念验证、因果论证、定量关系

## 实验室实验设计

### 体外实验

**何时使用：** 测试受控系统中的分子、细胞或生化机制。

**常见模式：**

#### 1. 剂量反应研究
- **目的：**建立输入与效果之间的定量关系
- **设计：** 测试多种浓度/剂量的干预
- **关键要素：**
  - 阴性对照（不治疗）
  - 阳性对照（已知有效治疗）
  - 多个剂量水平（通常为 5-8 点）
  - 技术重复（每个条件≥3）
  - 适当的统计分析（曲线拟合、IC50/EC50测定）

**应用示例：**
“为了测试化合物 X 是否抑制酶 Y，请测量 0、1、10、100、1000 nM 化合物 X 浓度下的酶活性，每剂量重复 3 次。”

#### 2. 功能获得/丧失研究
- **目的：** 确定特定成分的因果作用
- **设计：** 添加（过度表达）或删除（敲除/敲除）组件
- **关键要素：**
  - 野生型对照
  - 功能获得条件（过度表达、组成型激活）
  - 功能丧失情况（敲除、敲低、抑制）
  - 拯救实验（恢复功能至丧失功能）
  - 衡量下游影响

**应用示例：**
“通过以下方式测试蛋白质 X 是否会导致表型 Y：(1) 敲除 X 并观察表型丧失，(2) 过度表达 X 并观察表型增强，(3) 通过 X 重新表达来挽救敲除。”

#### 3. 时间课程研究
- **目的：** 了解时间动态和事件顺序
- **设计：** 在多个时间点测量结果
- **关键要素：**
  - 时间 0 基线
  - 早期时间点（捕捉快速变化）
  - 中间时间点
  - 晚时间点（稳态）
  - 每个时间点都有足够的复制

**应用示例：**
“在刺激后 0、5、15、30、60、120 分钟测量蛋白质磷酸化，以确定峰值激活时间。”

### 体内实验

**何时使用：** 在整个生物体中测试假设以评估系统、生理或行为影响。

**常见模式：**

#### 4. 受试者间设计
- **目的：** 比较接受不同治疗的不同组
- **设计：** 将受试者随机分配到治疗组
- **关键要素：**
  - 随机分配到组
  - 适当的样本量（功效分析）
  - 对照组（媒介物、假手术或标准治疗）
  - 盲法（单盲或双盲）
  - 各组的标准化条件

**应用示例：**
“将 20 只小鼠随机分配到载体对照组或药物治疗组，每周测量肿瘤大小，持续 8 周，实验人员对分组情况不知情。”

#### 5. 受试者内（重复测量）设计
- **目的：** 每个受试者都作为自己的对照，减少受试者间的变异性
- **设计：** 在多个条件/时间点测量相同的受试者
- **关键要素：**
  - 基线测量
  - 平衡（如果可能产生订单效应）
  - 冲洗期（连续治疗）
  - 适当的重复测量统计

**应用示例：**
“在基线、训练干预后以及 3 个月的随访中测量相同参与者的认知表现。”

#### 6. 因子设计
- **目的：** 同时测试多个因素及其相互作用
- **设计：** 跨越多个自变量的所有水平
- **关键要素：**
  - 清晰的主要影响和相互作用
  - 足够的电量进行交互测试
  - 适当的全阶乘或分数阶乘

**应用示例：**
“2×2设计交叉基因型（WT与突变体）×治疗（载体与药物）来测试药物效果是否取决于基因型。”

### 计算/建模实验
**何时使用：** 测试有关复杂系统的假设、做出预测，或者当物理实验不可行时。

#### 7. 计算机模拟
- **目的：** 复杂系统建模，测试理论预测
- **设计：**实施计算模型并改变参数
- **关键要素：**
  - 具有明确假设的明确定义的模型
  - 参数敏感性分析
  - 根据已知数据进行验证
  - 实验测试的预测生成

**应用示例：**
“建立基于主体的疾病传播模型，改变传播率和干预时间，将预测与经验流行病数据进行比较。”

#### 8. 生物信息学/荟萃分析
- **目的：** 使用现有数据集测试假设
- **设计：** 分析大规模数据或汇总多项研究
- **关键要素：**
  - 适当的统计修正（多次测试）
  - 在独立数据集中进行验证
  - 控制混杂因素和批次效应
  - 明确的纳入/排除标准

**应用示例：**
“使用带有临床协变量的 Cox 回归来测试基因 X 表达是否与 15 个癌症数据集（总共 5000 名患者）的生存相关。”

## 观察性研究设计

### 当身体操纵不可能或不道德时

#### 9. 横断面研究
- **目的：** 检查单个时间点的关联
- **设计：** 一次性测量人群中感兴趣的变量
- **优点：** 快速、便宜、可以建立流行
- **限制：** 无法建立时间性或因果关系
- **关键要素：**
  - 代表性抽样
  - 标准化测量
  - 控制混杂变量
  - 适当的统计分析

**应用示例：**
“对 1000 名成年人进行调查，以测试饮食模式与生物标志物 X 之间的关联，并控制年龄、性别、体重指数和体力活动。”

#### 10. 队列研究（前瞻性/纵向）
- **目的：** 建立时间关系和潜在的因果关系
- **设计：** 随着时间的推移跟踪小组，衡量暴露和结果
- **优点：** 可以建立时间性，计算发生率
- **限制：** 耗时、昂贵、受试者流失
- **关键要素：**
  - 基线暴露评估
  - 按规定的时间间隔进行跟进
  - 最大限度地减少后续损失
  - 考虑随时间变化的混杂因素

**应用示例：**
“对 5000 名最初健康的人进行为期 10 年的跟踪，测试基线维生素 D 水平是否可以预测心血管疾病的发病率。”

#### 11.病例对照研究
- **目的：** 通过比较病例与对照，有效研究罕见结果
- **设计：** 识别具有结果的病例，选择匹配的对照，比较暴露情况
- **优点：** 对罕见疾病有效，相对较快
- **局限性：**回忆偏差、选择偏差、无法计算发生率
- **关键要素：**
  - 清晰的案例定义
  - 适当的控制选择（匹配或统计调整）
  - 回顾性暴露评估
  - 混杂控制

**应用示例：**
“将 200 名患有罕见疾病 X 的患者与 400 名没有 X 的匹配对照患者进行比较，测试两组之间的早期暴露 Y 是否存在差异。”

## 临床试验设计

#### 12. 随机对照试验 (RCT)
- **目的：** 测试人类干预措施的黄金标准
- **设计：** 随机分配参与者进行治疗或对照
- **关键要素：**
  - 随机化（简单、分组或分层）
  - 隐藏分配
  - 盲法（参与者、提供者、评估者）
  - 意向治疗分析
  - 预先注册的方案和分析计划

**应用示例：**
“双盲随机对照试验：随机分配 300 名患者接受 X 药物或安慰剂治疗 12 周，测量症状改善的主要结果。”

#### 13. 交叉试验
- **目的：** 每个参与者按顺序接受所有治疗
- **设计：** 参与者在不同治疗之间交叉进行洗脱
- **优点：** 减少受试者间的变异性，需要更少的参与者
- **限制：** 顺序效应，需要可逆条件，持续时间较长
- **关键要素：**
  - 足够的冲洗期
  - 随机治疗顺序
  - 残留效应评估

**应用示例：**
“交叉试验：参与者接受治疗 A 4 周，2 周清除，然后接受治疗 B 4 周（随机顺序）。”

## 高级设计考虑因素

### 样本量和统计功效

**关键问题：**
- 多大的效应量才有意义？
- 将使用什么统计测试？
- 什么 alpha（显着性水平）和 beta（功效）合适？
- 测量的预期变化是什么？

**一般准则：**
- 实验前进行正式的功率分析
- 对于试点研究，每组最少 n≥10
- 对于确定性研究，目标为 ≥80% 功效
- 考虑纵向研究中的潜在流失

### 控制

**控制类型：**
- **阴性对照：** 无干预（基线）
- **积极控制：** 已知的有效干预（验证系统）
- **车辆控制：** 无活性成分的递送方法
- **假手术控制：** 模拟干预，无需活性成分（手术等）
- **历史控制：** 先前数据（最弱，尽可能避免）

### 致盲

**级别：**
- **开放标签：** 无盲法（客观测量可接受）
- **单盲：** 参与者失明（减少安慰剂效应）
- **双盲：** 参与者和实验者不知情（减少评估偏差）
- **三盲：** 参与者、实验者和分析师均盲（最强）

### 复制

**技术重复：** 对同一样品进行重复测量
- 减少测量误差
- 通常 2-3 个重复就足够了

**生物复制：** 独立样本/受试者
- 解决生物变异性
- 对于泛化至关重要
- 最少：每组n≥3，最好n≥5-10

**实验重复：**重复整个实验
- 跨时间、设备、操作员验证结果
- 确认结果的黄金标准

### 混淆控制

**策略：**
- **随机化：** 在各组之间均匀分布混杂因素
- **匹配：** 在不同条件下将相似的受试者配对
- **分块：** 按混淆分组，然后在块内随机化
- **统计调整：** 测量混淆并在分析中进行调整
- **标准化：** 保持各组条件恒定

## 选择合适的设计

**决策树：**

1. **变量可以被操纵吗？**
   - 是 → 实验设计（RCT、实验室实验）
   - 否 → 观察设计（队列、病例对照、横断面）

2. **系统是什么？**
   - 细胞/分子→体外实验
   - 整个生物体 → 体内实验
   - 人类 → 临床试验或观察性研究
   - 复杂系统→计算建模

3. **主要目标是什么？**
   - 机制→功能获得/丧失，剂量反应
   - 因果关系 → RCT，具有良好对照的队列研究
   - 协会 → 横断面、病例对照
   - 预测→建模、机器学习
   - 时间动态 → 时间过程，纵向

4. **有什么限制？**
   - 时间有限 → 横截面，体外
   - 预算有限 → 计算、观察
   - 伦理问题 → 观察、体外
   - 罕见结果→病例对照、荟萃分析

## 整合多种方法

强有力的假设检验通常结合多种设计：

**示例：测试微生物组是否影响认知功能**
1. **观察性：** 队列研究显示微生物组组成与认知之间的关联
2. **动物模型：**接受微生物组移植的无菌小鼠表现出认知变化
3. **机制：** 体外研究显示微生物代谢物影响神经元功能
4. **临床试验：** 益生菌干预改善认知评分的随机对照试验
5. **计算：** 预测哪些微生物组特征会影响认知的模型

**三角测量方法：**
- 每个设计解决不同的方面/限制
- 来自多种方法的汇聚证据强化了因果关系主张
- 从观察/体外开始，然后进行明确的因果测试

## 常见陷阱

- 样本量不足（动力不足）
- 缺乏适当的控制
- 未考虑混杂变量
- 不适当的统计测试
- P-hacking 或未经修正的多次测试
- 涉及主观评估时缺乏盲法
- 未能复制研究结果
- 未预先注册分析计划（临床试验）

## 假设检验的实际应用

在设计实验来检验假设时：

1. **匹配设计与假设细节：** 因果关系需要实验操作；协会可以使用观察性设计
2. **开始简单，然后详细：** 先进行简单设计，然后增加复杂性
3. **仔细规划控制：** 控制验证系统并隔离特定效果
4. **考虑可行性：**平衡理想设计与实际约束
5. **计划多项实验：** 很少有一项实验能够明确检验一个假设
6. **预先指定分析：** 在数据收集之前决定统计测试
7. **内置验证：** 独立复制、正交方法、收敛证据
