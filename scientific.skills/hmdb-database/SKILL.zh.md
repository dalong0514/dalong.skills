# HMDB 数据库

## 概述

人类代谢组数据库 (HMDB) 是一个全面、免费的资源，包含有关人体内发现的小分子代谢物的详细信息。

## 何时使用此技能

在执行代谢组学研究、临床化学、生物标志物发现或代谢物识别任务时应使用此技能。

## 数据库内容

HMDB 版本 5.0（截至 2025 年）包含：

- **220,945 个代谢物条目** 涵盖水溶性和脂溶性化合物
- **8,610 个蛋白质序列**，涉及代谢中涉及的酶和转运蛋白
- **每个代谢物 130 多个数据字段** 包括：
  - 化学性质（结构、分子式、分子量、InChI、SMILES）
  - 临床数据（生物标志物关联、疾病、正常/异常浓度）
  - 生物信息（途径、反应、位置）
  - 光谱数据（NMR、MS、MS-MS 光谱）
  - 外部数据库链接（KEGG、PubChem、MetaCyc、ChEBI、PDB、UniProt、GenBank）

## 核心能力

### 1. 基于网络的代谢物搜索

通过位于 https://www.hmdb.ca/ 的 Web 界面访问 HMDB：

**文本搜索：**
- 按代谢物名称、同义词或标识符（HMDB ID）搜索
- HMDB ID 示例：HMDB0000001、HMDB0001234
- 按疾病关联或通路参与搜索
- 按生物样本类型查询（尿液、血清、脑脊液、唾液、粪便、汗液）

**基于结构的搜索：**
- 使用 ChemQuery 进行结构和子结构搜索
- 按分子量或分子量范围搜索
- 使用 SMILES 或 InChI 字符串查找化合物

**光谱搜索：**
- LC-MS光谱匹配
- GC-MS 光谱匹配
- NMR 光谱搜索用于代谢物鉴定

**高级搜索：**
- 结合多个标准（名称、特性、浓度范围）
- 按生物位置或样本类型过滤
- 按蛋白质/酶关联搜索

### 2. 访问代谢物信息

检索代谢物数据时，HMDB 提供：

**化学信息：**
- 系统名称、传统名称和同义词
- 化学式和分子量
- 结构表示（2D/3D、SMILES、InChI、MOL 文件）
- 化学分类学和分类

**生物学背景：**
- 代谢途径和反应
- 相关酶和转运蛋白
- 亚细胞位置
- 生物学作用和功能

**临床相关性：**
- 生物体液中的正常浓度范围
- 生物标志物与疾病的关联
- 临床意义
- 适用时的毒性信息

**分析数据：**
- 实验和预测的核磁共振谱
- MS 和 MS-MS 谱图
- 保留时间和色谱数据
- 用于识别的参考峰

### 3.可下载的数据集

HMDB 在 https://www.hmdb.ca/downloads 提供多种格式的批量数据下载：

**可用格式：**
- **XML**：完整的代谢物、蛋白质和光谱数据
- **SDF**：化学信息学的代谢物结构文件
- **FASTA**：蛋白质和基因序列
- **TXT**：原始光谱峰值列表
- **CSV/TSV**：表格数据导出

**数据集类别：**
- 所有代谢物或按样本类型过滤
- 蛋白质/酶序列
- 实验和预测光谱（NMR、GC-MS、MS-MS）
- 途径信息

**最佳实践：**
- 下载XML格式的全面数据，包括所有领域
- 使用 SDF 格式进行基于结构的分析和化学信息学工作流程
- 解析 CSV/TSV 格式以与数据分析管道集成
- 检查版本日期以确保最新数据（当前：v5.0，2023-07-01）

**使用要求：**
- 免费用于学术和非商业研究
- 商业用途需要明确许可（联系 samackay@ualberta.ca）
- 使用数据时引用 HMDB 出版物

### 4. 编程式 API 访问

**API 可用性：**
HMDB 不提供公共 REST API。编程访问需要联系开发团队：

- **学术/研究小组：** 联系 eponine@ualberta.ca (Eponine) 或 samackay@ualberta.ca (Scott)
- **商业组织：** 联系 samackay@ualberta.ca (Scott) 以获取定制 API 访问权限

**替代编程访问：**
- **R/Bioconductor**：使用 `hmdbQuery` 包进行基于 R 的查询
  - 安装：`BiocManager::install("hmdbQuery")`
- 提供基于HTTP的查询功能
- **下载的数据集**：在本地解析 XML 或 CSV 文件以进行编程分析
- **网页抓取**：不推荐；请联系团队以获得正确的 API 访问权限

### 5. 通用研究工作流程

**非靶向代谢组学中的代谢物鉴定：**
1. 从样品中获取实验MS或NMR谱
2. 使用HMDB光谱搜索工具与参考光谱进行匹配
3. 通过检查分子量、保留时间和 MS-MS 碎片来验证候选物
4. 审查生物学合理性（预期样本类型、已知途径）

**生物标志物发现：**
1. 在 HMDB 中搜索与感兴趣的疾病相关的代谢物
2. 检查正常与疾病状态下的浓度范围
3. 识别丰度差异较大的代谢物
4. 检查通路背景和生物学机制
5. 通过 PubMed 链接与文献交叉引用

**路径分析：**
1. 从实验数据中识别感兴趣的代谢物
2. 查找每个代谢物的 HMDB 条目
3. 提取途径关联和酶促反应
4. 使用链接的 SMPDB（小分子通路数据库）绘制通路图
5. 确定生物学解释的途径富集

**数据库集成：**
1. 下载XML或CSV格式的HMDB数据
2.解析并提取本地数据库的相关字段
3. 与外部ID（KEGG、PubChem、ChEBI）链接以进行跨数据库查询
4. 构建包含 HMDB 参考数据的本地工具或管道

## 相关 HMDB 资源

HMDB生态系统包括相关数据库：

- **DrugBank**：约 2,832 种药物化合物以及药物信息
- **T3DB（毒素和毒素目标数据库）**：约 3,670 种有毒化合物
- **SMPDB（小分子通路数据库）**：通路图和地图
- **FooDB**：约 70,000 种食品成分化合物

这些数据库具有相似的结构和标识符，可以跨人类代谢组、药物、毒素和食品数据库进行集成查询。

## 最佳实践

**数据质量：**
- 使用多种证据类型（光谱、结构、性质）验证代谢物鉴定
- 检查实验与预测数据质量指标

- 注意研究中使用的 HMDB 版本（当前：v5.0）
- 数据库定期更新新条目和更正
- 发布时重新查询更新，确保信息最新

**引用：**
- 始终在使用数据库的出版物中引用 HMDB
- 讨论代谢物时参考特定的 HMDB ID
- 确认下载数据集的数据源

**性能：**
- 对于大规模分析，下载完整的数据集而不是重复的网络查询
- 使用适当的文件格式（XML用于综合数据，CSV用于表格分析）
- 考虑对经常访问的代谢物信息进行本地缓存
