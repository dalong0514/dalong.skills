# 科学手稿中常见的方法论和统计问题

本文档按类别整理了同行评审过程中遇到的常见问题。以此作为参考来识别潜在问题并提供建设性反馈。

## 统计问题

### 1. P 值误用和误解

**常见问题：**
- P-hacking（选择性报告重要结果）
- 多次测试，无需修正（家庭错误率膨胀）
- 将无意义解释为无效的证明
- 只关注 p 值而不考虑效应大小
- 在任意阈值处二分连续 p 值（p=0.049 与 p=0.051）
- 混淆统计显着性与生物学/临床显着性

**如何识别：**
- p 值略低于 0.05 的比例过高，令人怀疑
- 进行了许多测试，但没有提到纠正
- 诸如“没有发现差异”之类的陈述与不显着的结果
- 没有报告效应大小或置信区间
- 建议 p 值表明效果强度的语言

**推荐什么：**
- 报告效应大小和置信区间
- 应用适当的多重测试修正（Bonferroni、FDR、Holm-Bonferroni）
- 谨慎解释不显着性（缺乏证据≠缺乏证据）
- 预先注册分析以避免 p-hacking
- 考虑对“无差异”声明进行等价测试

### 2. 不适当的统计测试

**常见问题：**
- 在违反假设时使用参数检验（非正态数据、不等方差）
- 通过不配对测试分析配对数据
- 对多个组使用 t 检验，而不是使用事后检验的方差分析
- 将序数数据视为连续数据
- 忽略重复测量结构
- 当回归更合适时使用相关性

**如何识别：**
- 没有提到假设检查
- 小样本量的参数测试
- 多个成对 t 检验而不是方差分析
- 通过 t 检验分析李克特量表
- 分析时间序列数据而不考虑重复测量

**推荐什么：**
- 明确检查假设（正态性检验、Q-Q 图）
- 适当时使用非参数替代方案
- 方差分析后对多重比较应用适当的修正
- 使用混合效应模型进行重复测量
- 考虑序数结果的序数回归

### 3. 样本量和功效问题

**常见问题：**
- 没有样本量理由或功效计算
- 动力不足的研究声称“没有效果”
- 事后功率计算（无信息）
- 未预先指定停止规则
- 没有理由的不平等的团体规模

**如何识别：**
- 小样本量（典型设计每组 n<30）
- 方法中没有提及功效分析
- 关于事后权力的陈述
- 宽置信区间表明不精确
- 声称“没有效果”，p 值较大且 n 较小

**推荐什么：**
- 根据预期效果大小进行先验功效分析
- 报告达到的功效或精度（置信区间宽度）
- 当研究动力不足时予以承认
- 考虑效应大小和置信区间进行解释
- 预先登记样本大小和停止规则

### 4. 缺失数据问题

**常见问题：**
- 没有理由的完整案例分析（列表删除）
- 不报告缺失的程度或模式
- 假设数据完全随机丢失（MCAR）而无需测试
- 不当的插补方法
- 不进行敏感性分析

**如何识别：**
- 不同分析中的 n 值不同，无需解释
- 不讨论缺失数据
- 参与者“被排除在分析之外”
- 使用简单平均插补
- 没有比较完整数据与估算数据的敏感性分析

**推荐什么：**
- 报告缺失的程度和模式
- 测试 MCAR 假设（Little 测试）
- 使用适当的方法（多重插补、最大似然）
- 进行敏感性分析
- 考虑试验的意向治疗分析

### 5.循环分析和双浸

**常见问题：**
- 使用相同的数据进行选择和推理
- 根据对比度定义 ROI，然后在同一 ROI 中测试对比度
- 选择异常值然后测试差异
- 按计划提出事后亚组分析
- HARKing（已知结果后进行假设）

**如何识别：**
- 根据结果选择的 ROI 或功能
- 意外的亚组分析
- 事后分析未明确标记为探索性
- 没有独立于数据的验证
- 完美预测结果的介绍

**推荐什么：**
- 使用独立的数据集进行选择和测试
- 预注册分析和假设
- 明确区分验证性分析与探索性分析
- 使用交叉验证或保留数据集
- 纠正选择偏差

### 6. 伪复制

**常见问题：**
- 技术复制被视为生物复制
- 来自同一受试者的多次测量被视为独立
- 在不考虑聚类的情况下分析聚类数据
- 空间或时间数据的非独立性

**如何识别：**
- n 定义为测量次数而不是生物单位
- 来自同一动物的多个细胞被视为独立的
- 重复措施未获承认
- 没有提及随机效应或聚类

**推荐什么：**
- 将 n 定义为生物重复（动物、患者、独立样本）
- 对嵌套或集群数据使用混合效应模型
- 明确考虑重复措施
- 分析前的平均技术重复
- 报告技术和生物复制

## 实验设计问题

### 7. 缺乏适当的控制

**常见问题：**
- 缺少阴性对照
- 缺少验证的阳性对照
- 药物研究没有媒介物控制
- 纵向研究没有时间匹配的对照
- 无批次控制

**如何识别：**
- 方法部分仅列出实验组
- 图中未提及控件
- 基线或参考条件不明确
- 无对照的跨批次比较

**推荐什么：**
- 包括阴性对照以评估特异性
- 包括阳性对照以验证方法
- 使用与实验治疗相匹配的载体对照
- 包括手术干预的假手术对照
- 包括用于跨批次比较的批次控制

### 8. 混杂变量

**常见问题：**
- 除干预外，组间存在系统性差异
- 批次效应未得到控制或纠正
- 顺序实验中的顺序效应
- 时间效应不受控制
- 实验者的影响不致盲

**如何识别：**
- 群体在多个特征上存在差异
- 按组分批处理的样品
- 样品顺序没有随机化
- 没有提到致盲
- 各组的基线特征不同

**推荐什么：**
- 根据条件随机化实验单位
- 阻止已知的混杂因素
- 随机化样本处理顺序
- 使用盲法来最大限度地减少偏见
- 如果需要，执行批量校正
- 报告并调整基线差异

### 9. 复制不足

**常见问题：**
- 单一实验，无需重复
- 技术复制被误认为是生物复制
- 小n的理由是“该领域的典型”
- 没有对关键发现进行独立验证
- 挑选代表性例子

**如何识别：**
- 方法声明“实验进行一次”
- n=3 没有理由
- “显示代表性图像”
- 基于单一实验的关键主张
- 没有在独立数据集中进行验证

**推荐什么：**
- 进行独立的生物学重复（通常≥3）
- 验证独立队列中的关键发现
- 报告所有重复，而不仅仅是代表性示例
- 进行功效分析以证明样本量的合理性
- 显示单个数据点，而不仅仅是汇总统计数据

## 再现性问题

### 10.方法细节不足

**常见问题：**
- 方法描述不够详细，无法复制
- 未指定关键试剂（供应商、目录号）
- 未报告软件版本和参数
- 抗体未经验证
- 细胞系验证未验证

**如何识别：**
- 模糊的描述（“使用了标准协议”）
- 没有试剂来源信息
- 提到的通用软件没有版本
- 没有抗体验证信息
- 细胞系未经验证

**推荐什么：**
- 提供详细的协议或引用特定的协议
- 包括试剂供应商、目录号、批号
- 报告软件版本和所有参数
- 包括抗体验证（蛋白质印迹、特异性测试）
- 报告细胞系验证方法（STR分析）
- 提供协议（protocols.io，补充材料）

### 11. 数据和代码可用性

**常见问题：**
- 无数据可用性声明
- “根据要求提供数据”（通常未实现）
- 没有提供用于计算分析的代码
- 不提供定制软件
- 没有明确的文档

**如何识别：**
- 缺少数据可用性声明
- 没有存储库登录号
- 无需代码的计算方法
- 无需访问的自定义管道
- 没有自述文件或文档

**推荐什么：**
- 将原始数据存入适当的存储库（GEO、SRA、Dryad、Zenodo）
- 在 GitHub 或类似平台上分享分析代码
- 提供清晰的文档和自述文件
- 包含requirements.txt或环境文件
- 提供带有安装说明的定制软件
- 使用 DOI 进行永久数据引用

### 12.缺乏方法验证

**常见问题：**
- 新方法未与黄金标准进行比较
- 检测方法未经过特异性、敏感性、线性验证
- 无尖峰控制
- 未测试交叉反应性
- 未建立检测限

**如何识别：**
- 未经验证而提出的新颖检测方法
- 无法与现有方法进行比较
- 未显示阳性/阴性对照
- 没有证据的特殊性主张
- 没有标准曲线或对照

**推荐什么：**
- 根据既定方法验证新方法
- 显示特异性（敲除/敲除对照）
- 展示线性度和动态范围
- 包括阳性和阴性对照
- 报告检测和定量的限度
- 显示重复和操作员之间的再现性

## 解释问题

### 13. 夸大结果

**常见问题：**
- 相关数据的因果语言
- 没有机械证据的机械主张
- 超越数据（物种、条件、种群）进行推断
- 未经彻底的文献综述就声称“首先展示”
- 从有限的样本中过度概括

**如何识别：**
- 观察数据中的“X导致Y”
- 未经直接测试而提出的机制
- 鼠标数据呈现为与人类相关，没有任何警告
- 缺少引用的新颖性声明
- 来自狭窄样本的广泛主张

**推荐什么：**
- 使用适当的语言（“与……相关”与“由……引起”）
- 区分相关性和因果关系
- 承认模型系统的局限性
- 提供详尽的文献背景
- 明确普遍性
- 提出机制作为假设，而不是结论

### 14. 精挑细选和选择性报告

**常见问题：**
- 仅报告重要结果
- 显示可能不典型的“代表性”图像
- 排除无正当理由的异常值
- 不报告负面或矛盾的发现
- 不同统计方法之间的切换

**如何识别：**
- 所有报告的结果都很重要
- 没有量化的“3个实验的代表”
- 结果中提到的数据排除，但方法中没有提到
- 补充数据与主要发现相矛盾
- 多种分析方法，仅报告一种

**推荐什么：**
- 报告所有计划的分析，无论结果如何
- 量化并显示重复的变异性
- 预先指定异常值排除标准
- 包括负面结果
- 预登记分析计划
- 报告所有比较的效果大小和置信区间

### 15. 忽略其他解释

**常见问题：**
- 在不考虑替代方案的情况下提出首选解释
- 未经讨论就驳回矛盾证据
- 未考虑脱靶效应
- 未承认混杂变量
- 限制部分最少或不存在

**如何识别：**
- 作为事实呈现的单一解释
- 未引用或讨论先前的矛盾发现
- 不考虑替代机制
- 不讨论限制
- 假设没有对照的特异性

**推荐什么：**
- 讨论替代解释
- 解决文献中相互矛盾的发现
- 包括适当的特异性控制
- 彻底承认并讨论局限性
- 考虑并测试替代假设

## 图形和数据表示问题

### 16. 不恰当的数据可视化

**常见问题：**
- 连续数据的条形图（隐藏分布）
- 无误差线或未定义误差线
- 截断的 y 轴夸大了差异
- 双 y 轴造成误导性比较
- 有效数字过多
- 颜色不适合色盲

**如何识别：**
- 数据点很少的条形图
- 不清楚误差条代表什么（SD、SEM、CI？）
- 对于比率/百分比数据，Y 轴不从零开始
- 左右 y 轴具有不同的比例
- 报告的值精度过高 (p=0.04562)
- 红绿配色方案

**推荐什么：**
- 用散点图/箱线图/小提琴图显示各个数据点
- 始终定义误差线（SD、SEM、95% CI）
- 从零开始 y 轴或清楚地指示中断点
- 避免双 y 轴；使用单独的面板代替
- 报告适当的有效数字
- 使用色盲友好的调色板（viridis、colorbrewer）
- 在图例中包含样本大小

### 17. 图像处理问题

**常见问题：**
- 过度的对比度/亮度调整
- 无指示的拼接凝胶或图像
- 重复的图像或面板
- 蛋白质印迹中背景不均匀
- 选择性裁剪
- 过度处理的显微镜图像

**如何识别：**
- 可疑的模式或不连续性
- 对比度非常高，没有背景
- 不同面板中的类似功能
- 直线暗示拼接
- 背景不一致
- 细节丢失表明过度处理

**推荐什么：**
- 在图像上统一应用调整
- 用分割线指示拼接凝胶
- 在补充材料中显示完整的、未裁剪的图像
- 如果需要，提供原始图像
- 遵循期刊形象完整性政策
- 使用适当的图像分析工具

## 研究设计问题

### 18. 假设和结果定义不明确

**常见问题：**
- 没有明确的假设
- 主要结果未指定
- 无需修正的多种结果
- 数据收集后结果发生变化
- 钓鱼探险呈现为假设驱动的

**如何识别：**
- 引言没有陈述明确的可检验假设
- 层次不明确的多种结果
- 结果中的结果与方法中的结果不匹配
- 作为验证性的探索性研究
- 多次测试，无需多次测试修正

**推荐什么：**
- 陈述清晰、可检验的假设
- 先验指定主要和次要结果
- 尽可能预先注册研究
- 对多种结果应用适当的修正
- 明确区分探索性分析和验证性分析
- 报告所有预先指定的结果

### 19. 基线不平衡和选择偏差

**常见问题：**
- 各组的基线不同
- 不同的选择标准
- 健康的志愿者偏见
- 幸存者偏差
- 观察性研究中的指示偏差

**如何识别：**
- 表 1 显示显着的基线差异
- 组间的纳入标准不同
- 响应率 <50%，无需分析
- 分析仅包括完成者
- 小组自行选择而非随机

**推荐什么：**
- 报告表1中的基线特征
- 使用随机化来确保平衡
- 调整分析中的基线差异
- 报告响应率并比较响应者与非响应者
- 考虑观察数据的倾向得分匹配
- 使用意向治疗分析

### 20. 时间和批次效应

**常见问题：**
- 按条件批量处理样品
- 未考虑时间趋势
- 仪器随时间的漂移
- 不同的运营商针对不同的群体
- 组间试剂批次变化

**如何识别：**
- 所有处理样品均在同一天处理
- 不同时间段的控制
- 没有提及批次或时间效应
- 团体不同的技术人员
- 研究持续时间长，无时间分析

**推荐什么：**
- 跨批次/时间随机化样本
- 在分析中包含批次作为协变量
- 执行批量校正（ComBat、limma）
- 包括跨批次的质量控制样品
- 报告和测试时间趋势
- 平衡不同条件下的操作员

## 报告问题

### 21. 统计报告不完整

**常见问题：**
- 未报告测试统计数据
- 缺少自由度
- 精确的 p 值替换为不等式 (p<0.05)
- 无置信区间
- 无效应大小
- 未报告每组的样本量

**如何识别：**
- 仅给出 p 值，没有测试统计数据
- p 值报告为 p<0.05，而不是精确值
- 没有不确定性的衡量标准
- 影响程度尚不清楚
- n 报告总数，但不按组报告

**推荐什么：**
- 报告完整的测试统计数据（t、F、χ2 等与 df）
- 报告精确的 p 值（p<0.001 除外）
- 包括 95% 置信区间
- 报告效应大小（Cohen's d、比值比、相关系数）
- 在每次分析中为每个组报告 n
- 考虑 CONSORT 式流程图

### 22.方法-结果不匹配

**常见问题：**
- 方法描述了未执行的分析
- 结果包括方法中未描述的分析
- 方法与结果的样本量不同
- 方法提及未显示的控件
- 统计方法与实际情况不符

**如何识别：**
- 没有方法描述的结果分析
- 方法描述的是实验而不是结果
- 各部分之间的数字不匹配
- 提到但未显示的控件
- 提到的软件与使用的软件不同

**推荐什么：**
- 确保方法和结果完全一致
- 描述方法中进行的所有分析
- 删除未进行的实验的方法描述
- 验证所有数字是否一致
- 更新方法以匹配实际进行的分析

## 如何使用此参考
