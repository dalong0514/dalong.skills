# Paper2Poster：学术海报生成

## 概述

Paper2Poster 自动从研究论文生成专业的学术海报。该系统提取关键内容，设计具有视觉吸引力的布局，并创建适合会议、研讨会和学术演讲的可打印海报。

## 核心能力

### 1.内容提取
- 确定主要发现和贡献
- 提取重要的数字和表格
- 总结方法论
- 强调结果和结论
- 保留引文和参考文献

### 2.布局设计
- 创建平衡、专业的布局
- 优化内容密度和空白区域
- 建立清晰的视觉层次结构
- 支持多种海报尺寸
- 适应不同的内容类型

### 3.视觉设计
- 应用配色方案和品牌
- 优化排版以提高可读性
- 确保身材质量和尺寸
- 创造有凝聚力的视觉形象
- 维持学术表现标准

### 参数

**基本配置：**
- `--input-dir`：包含纸质文件的目录
- `--output-dir`：生成海报的目录
- `--model-choice`：LLM模型选择（1=GPT-4，2=GPT-4.1）
- `--generate-poster`：启用海报生成

**海报尺寸：**
- `--poster-width-inches`：宽度（以英寸为单位）（默认值：48）
- `--poster-height-inches`：高度（以英寸为单位）（默认值：36）
- `--poster-orientation`：纵向或横向（默认：横向）
- `--poster-dpi`：DPI 分辨率（默认值：300）

**设计选项：**
- `--poster-template`：模板样式（默认：现代）
- `--color-scheme`：调色板选择
- `--institution-branding`：包括机构颜色和徽标
- `--font-family`：版式选择

## 打印准备

### 文件规范
- **格式**：用于专业打印的 PDF/X-1a 或 PDF/X-4
- **分辨率**：最低 300 DPI，精细细节 600 DPI
- **颜色模式**：用于打印的 CMYK（系统自动从 RGB 转换）
- **出血**：所有侧面 0.125 英寸出血（自动添加）
- **字体**：PDF 中嵌入的所有字体

### 打印建议
1. **打印店**：使用专业的海报打印服务
2. **纸张类型**：用于学术海报的哑光或缎面处理
3. **背衬**：泡沫芯或刚性背衬以确保稳定性
4. **保护**：层压可选，但推荐
5. **测试打印**：先打印A4/Letter尺寸预览

### 预算选项
- **标准**：在专业商店购买 4'×3' 海报 50-100 美元
- **经济**：仅打印版（无装裱）20-40 美元
- **高级**：高端材料和安装费用为 150-300 美元
- **DIY**：多页平铺和组装 <10 美元

## 最佳实践

### 内容优化
1. **关注关键发现**：海报应该一目了然地讲述故事
2. **限制文本**：使用项目符号，避免段落
3. **优先考虑视觉效果**：人物应该占据空间的主导地位
4. **清晰的流程**：引导观看者完成逻辑进展
5. **突出贡献**：让新颖性变得明显

### 设计优化
1. **使用对比度**：确保文本易于阅读
2. **维护层次结构**：大小表明重要性
3. **平衡内容**：避免任何部分拥挤
4. **一致的样式**：始终使用相同的字体、颜色
5. **空白**：不要填满每一寸

### 身材优化
1. **足够大**：主要人物的宽度至少为 6 英寸
2. **高分辨率**：最低 300 DPI
3. **清晰标签**：轴标签、图例可读
4. **消除混乱**：简化海报格式
5. **使用标题**：简短、信息丰富的描述

## 限制

- 复杂的方程可能需要手动调整以提高可读性
- 很长的论文可能需要内容优先顺序
- 自定义品牌需要手动规范或 API 访问
- 多语言支持仅限于通用语言
- 3D 可视化可能会降低 2D 海报格式的质量
