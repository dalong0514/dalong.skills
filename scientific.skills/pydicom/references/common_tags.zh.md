## DICOM 值表示 (VR)

DICOM 中使用的常见值表示类型：

- **AE**：应用程序实体（最多 16 个字符）
- **AS**：年龄字符串 (nnnD/W/M/Y)
- **CS**：代码字符串（最多 16 个字符）
- **DA**：日期 (YYYYMMDD)
- **DS**：十进制字符串
- **DT**：日期时间 (YYYYMMDDHHMMSS.FFFFFF&ZZXX)
- **IS**：整数字符串
- **LO**：长字符串（最多 64 个字符）
- **LT**：长文本（最多 10240 个字符）
- **PN**：人名
- **SH**：短字符串（最多 16 个字符）
- **SQ**：项目序列
- **ST**：短文本（最多 1024 个字符）
- **TM**：时间 (HHMMSS.FFFFFF)
- **UI**：唯一标识符 (UID)
- **UL**：无符号长整型（4 字节）
- **US**：无符号短整型（2 字节）
- **OB**：其他字节字符串
- **OW**：其他字符串
