# 实验类型和工作流程

## 概述

Adaptyv 提供多种实验分析类型以进行全面的蛋白质表征。每种实验类型都有特定的应用程序、工作流程和数据输出。

## 结合测定

### 描述

使用生物层干涉测量 (BLI) 测量蛋白质与靶标的相互作用，这是一种实时监测生物分子结合的无标记技术。

### 用例

- 抗体-抗原结合表征
- 受体-配体相互作用分析
- 蛋白质-蛋白质相互作用研究
- 亲和力成熟筛选
- 表位分箱实验

### 技术：生物层干涉测量法 (BLI)

BLI 测量两个表面反射光的干涉图样：
- **参考层** - 生物传感器尖端表面
- **生物层** - 累积的结合分子

当分子结合时，光学厚度增加，导致与结合成比例的波长偏移。

**优点：**
- 无标记检测
- 实时动力学
- 高通量兼容
- 适用于粗样品
- 最少的样品消耗

### 测量参数

**动力学常数：**
- **KD** - 平衡解离常数（结合亲和力）
- **kon** - 结合速率常数（结合速度）
- **koff** - 解离速率常数（解离速度）

**典型范围：**
- 强结合剂：KD < 1 nM
- 中等结合剂：KD = 1-100 nM
- 弱结合剂：KD > 100 nM

### 工作流程

1. **序列提交** - 提供FASTA格式的蛋白质序列
2. **表达** - 在适当的宿主系统中表达的蛋白质
3. **纯化** - 自动纯化方案
4. **BLI 测定** - 针对指定目标的实时结合测量
5. **分析** - 动力学曲线拟合和质量评估
6. **结果交付** - 将参数与置信度指标绑定

### 示例要求

- 蛋白质序列（标准氨基酸代码）
- 目标规格（来自目录或定制请求）
- 缓冲条件（标准或定制）
- 预期浓度范围（可选，改进测定设计）

## 表达测试

### 描述

量化各种宿主系统中的蛋白质表达水平，以评估可生产性并优化制造序列。

### 用例

- 筛选高表达变体
- 优化密码子使用
- 识别表达瓶颈
- 选择扩大规模的候选人
- 比较表达系统

### 主机系统

可用的表达平台：
- **E。大肠杆菌** - 快速、经济高效的原核系统
- **哺乳动物细胞** - 天然翻译后修饰
- **酵母** - 具有更简单生长要求的真核系统
- **昆虫细胞** - 替代真核平台

### 测量参数

- **总蛋白产量**（mg/L 培养物）
- **可溶分数**（百分比）
- **纯度**（初步纯化后）
- **表达时间课程**（可选）

### 工作流程

1. **序列提交** - 提供蛋白质序列
2. **构建生成** - 克隆到表达载体中
3. **表达** - 指定宿主系统中的文化
4. **定量** - 通过多种方法测量蛋白质
5. **分析** - 表达水平比较和排名
6. **结果交付** - 产量数据和建议

## 热稳定性测试

### 描述

测量蛋白质热稳定性以评估结构完整性、预测保质期并识别稳定突变。

### 用例

- 选择热稳定变体
- 配方开发
- 保质期预测
- 稳定性驱动的蛋白质工程
- 质量控制筛选

### 测量技术

**差示扫描荧光测定法 (DSF)：**
- 通过荧光染料结合监测蛋白质的展开
- 确定熔化温度 (Tm)
- 高通量能力

**圆二色性 (CD)：**
- 二级结构分析
- 热展开曲线
- 可逆性评估

### 测量参数

- **Tm** - 熔化温度（展开的中点）
- **ΔH** - 展开焓
- **聚集温度** (Tagg)
- **可逆性** - 加热后重新折叠

### 工作流程

1. **序列提交** - 提供蛋白质序列
2. **表达和纯化** - 标准方案
3. **热稳定性测定** - 温度梯度分析
4. **数据分析** - 曲线拟合和参数提取
5. **结果交付** - 具有排名的稳定性指标

## 酶活性测定

### 描述

测量酶功能，包括底物周转率、催化效率和抑制剂敏感性。

### 用例

- 筛选酶变体以提高活性
- 底物特异性分析
- 抑制剂测试
- pH 值和温度优化
- 机理研究

### 检测类型

**连续检测：**
- 显色底物
- 荧光底物
- 实时监控

**终点检测：**
- HPLC 定量
- 质谱分析
- 比色检测

### 测量参数

**动力学参数：**
- **kcat** - 周转数（催化速率常数）
- **KM** - 米氏常数（底物亲和力）
- **kcat/KM** - 催化效率
- **IC50** - 50% 抑制的抑制剂浓度

**活动指标：**
- 比活性（单位/毫克蛋白质）
- 相对活动与参考
- 底物特异性概况

### 工作流程

1. **序列提交** - 提供酶序列
2. **表达和纯化** - 针对活性保留进行优化
3. **活性测定** - 底物周转测量
4. **动力学分析** - Michaelis-Menten 拟合
5. **结果交付** - 动力学参数和排名

## 实验设计最佳实践

### 序列提交

1. **使用清晰的标识符** - 描述性地命名序列
2. **包括对照** - 提交野生型或参考序列
3. **批量相似变体** - 在单次提交中对相关序列进行分组
4. **验证序列** - 提交前检查错误

### 样本量

- **试点研究** - 5-10 个序列来测试可行性
- **文库筛选** - 50-500 个序列用于变异探索
- **重点优化** - 10-50 个序列进行微调
- **大规模活动** - 500 多个用于 ML 驱动设计的序列

### 质量控制

Adaptyv 包括自动化 QC 步骤：
- 检测前的表达验证
- 重复测量以确保可靠性
- 每批的阳性/阴性对照
- 结果的统计验证

### 时间表预期

**标准周转时间：** 从提交到结果大约 21 天

**时间表细分：**
- 构建生成：3-5 天
- 表达：5-7天
- 净化：2-3天
- 检测执行：3-5 天
- 分析和质量控制：2-3 天

**影响时间线的因素：**
- 自定义目标（添加 1-2 周）
- 新型检测方法开发（增加 2-4 周）
- 大批量（可能会增加 1 周）

### 成本优化

1. **批量提交** - 降低每个序列的成本
2. **标准目标** - 目录抗原更快/更便宜
3. **标准条件** - 定制缓冲液会增加成本
4. **计算预过滤** - 只提交有前途的候选者

## 组合实验类型

为了进行全面的蛋白质表征，请结合多种测定：

**治疗性抗体开发：**
1. 结合测定 → 鉴定高亲和力结合物
2. 表达测试→选择可制造的候选者
3. 热稳定性 → 确保配方稳定性

**酶工程：**
1. 活性测定→筛选改进的催化作用
2. 表达测试→确保可生产性
3. 热稳定性 → 验证工业稳健性

**顺序与并行：**
- **顺序** - 使用早期测定的结果来筛选候选者
- **并行** - 同时运行所有测定以获得更快的结果
