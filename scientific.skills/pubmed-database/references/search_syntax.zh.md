## 邻近搜索

搜索彼此指定距离内的术语。仅在标题、标题/摘要和隶属关系字段中可用。

**语法**：`"search terms"[field:~N]`
- N = 术语之间的最大单词数

**示例**：
```
"vitamin C"[Title:~3]           → vitamin within 3 words of C in title
"breast cancer screening"[TIAB:~5]  → terms within 5 words in title/abstract
```

## 搜索字段标签

字段标签将搜索限制为 PubMed 记录的特定部分。格式：`term[tag]`

### 作者搜索

|标签 |领域 |示例|
|-----|--------|---------|
| [澳大利亚] |作者 |史密斯·J[澳大利亚] |
| [1au] |第一作者 |琼斯 m[1au] |
| [拉斯托] |最后作者 |威尔逊·K[拉斯托] |
| [法乌] |作者全名 |史密斯·约翰·a[fau] |

**作者搜索注释**：
- 可搜索 2002 年以后的完整作者姓名
- 格式：姓氏 + 缩写（例如，`smith ja[au]`）
- 无需字段标签即可搜索，但[au]确保准确性

**公司作者**：
搜索组织作为作者：
```
world health organization[au]
```

### 标题和摘要

|标签 |领域 |示例|
|-----|--------|---------|
| [钛]|标题 |糖尿病[ti] |
| [ab] |摘要|治疗[ab] |
| [提卜] |标题/摘要|癌症筛查[tiab] |
| [TW] |文字 |心血管[tw] |

**注释**：
- [tw] 搜索标题、摘要和其他文本字段
- [tiab] 最常用于综合搜索

### 期刊信息

|标签 |领域 |示例|

|标签 |领域 |格式|示例|
|-----|--------|--------|---------|
| [dp]|出版日期 |年/月/日 | 2023[dp] |
| [编辑] |恩特雷兹日期 |年/月/日 | 2023/01/15[编辑] |

**日期范围**：

## 自动术语映射 (ATM)

当搜索没有字段标签时，PubMed 会自动：

1. **搜索 MeSH 翻译表**以匹配 MeSH 术语
2. **搜索期刊翻译表**以获取期刊名称
3. **搜索作者索引**作者姓名
4. **搜索全文** 剩余术语

**绕过ATM**：
- 使用双引号：`"breast cancer"`
- 使用字段标签：`breast cancer[tiab]`

**查看翻译**：
使用高级搜索查看 PubMed 如何在搜索详细信息框中翻译您的查询。

## 最佳实践

### 1. 开始广泛，然后缩小
从一般术语开始并添加特殊性：
```
diabetes                                 → too broad
diabetes mellitus type 2                 → better
diabetes mellitus type 2[mh] AND treatment[tiab] → more specific
```

### 2. 使用 OR 的同义词
包括替代术语：
```
heart attack OR myocardial infarction OR MI
```

### 3. 用 AND 组合概念
链接您的研究问题的不同方面：
```
(heart attack OR myocardial infarction) AND (aspirin OR acetylsalicylic acid) AND prevention
```

### 4. 利用 MeSH 术语
使用 MeSH 进行一致的索引：
```
diabetes mellitus[mh] AND hypertension[mh]
```

### 5. 有策略地使用过滤器
应用过滤器来优化结果：
- 最近研究的发表日期
- 特定研究设计的文章类型
- 无障碍文章的免费全文

### 6. 查看搜索详细信息
检查 PubMed 如何在高级搜索中解释您的搜索以确保准确性。

## 故障排除

### 结果太多
- 添加更多具体条款
- 使用字段标签来限制搜索范围
- 应用日期限制
- 使用文章类型过滤器
- 使用 AND 添加附加概念

### 结果太少
- 删除限制性条款
- 使用 OR 添加同义词
- 检查拼写和术语
- 删除字段标签以进行更广泛的搜索
- 扩大日期范围
- 删除过滤器

### 没有结果
- 使用 ESpell 检查拼写
- 尝试替代术语
- 删除字段标签
- 验证正确的数据库（PubMed 与 PMC）
- 扩大搜索范围

### 意想不到的结果
- 查看搜索详细信息以查看查询翻译
- 使用字段标签来防止自动术语映射
- 检查可能包含的常见同义词
- 通过附加限制条款进行细化
