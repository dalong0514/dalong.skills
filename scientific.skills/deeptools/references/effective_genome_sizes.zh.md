# 有效基因组大小

## 定义

有效基因组大小是指“可映射”基因组的长度，即可以通过测序读数唯一映射的区域。该指标对于许多 deepTools 命令的正确规范化至关重要。

## 为什么它很重要

- RPGC 规范化所需 (`--normalizeUsing RPGC`)
- 影响覆盖率计算的准确性
- 必须符合您的数据处理方法（过滤与未过滤的读取）

## 计算方法

1. **非N碱基**：基因组序列中非N核苷酸的计数
2. **唯一可映射性**：可以唯一映射的特定大小的区域（可以考虑编辑距离）

## 选择正确的值

**对于大多数分析：** 使用参考基因组的非 N 碱基方法值

**对于过滤后的数据：** 如果您应用严格的质量过滤器或删除多重映射读取，请考虑使用特定于读取长度的值

**当不确定时：** 使用保守的非 N 基值 - 它的适用范围更广

## 常用快捷键
