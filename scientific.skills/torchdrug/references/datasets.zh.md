## 选择数据集

### 按任务

**分子性质预测：**
- 从 BBBP 或 HIV 开始（中等大小，任务明确）
- 使用 QM9 获得量子特性
- 用于回归的 ESOL/FreeSolv

**蛋白质功能：**
- EnzymeCommission（明确定义的类别）
- GeneOntology（综合注释）

**药物安全：**
- Tox21（标准基准）
- ClinTox（临床相关性）

**基于结构：**
- PDBBind（蛋白质配体）
- ProteinNet（结构预测）

**知识图谱：**
- FB15k-237（标准基准）
- Hetionet（生物医学应用）

**一代：**
- ZINC250k（训练）
- QM9（具有属性）

**逆合成：**
- USPTO-50k（唯一选择）

## 最佳实践

1. **从小规模开始**：在扩展之前在小数据集上进行测试
2. **支架拆分**：用于现实药物发现评估
3. **平衡指标**：对于不平衡数据使用AUROC + AUPRC
4. **多次运行**：报告多个随机种子的平均值±标准差
5. **数据泄露**：小心预训练模型
6. **领域知识**：了解您的预测
7. **验证**：始终使用保留的测试集
8. **预处理**：标准化特征，处理缺失值
