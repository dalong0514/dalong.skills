# ScholarEval 评估框架

## 概述

本文件为 ScholarEval 框架的每个维度提供了详细的评估标准、评分标准和质量指标。对学术工作进行系统评估时使用这些标准。

---

## 维度 1：问题表述和研究问题

### 质量指标

**优秀（5）：**
- 研究问题具体、可衡量且表述清晰
- 问题解决了具有高影响力潜力的文献中的重大差距
- 范围在限制范围内是适当且可行的
- 新颖的贡献与现有的工作明显不同
- 理论或实践意义令人信服

**好（4）：**
- 研究问题很清楚，但有一些模糊之处
- 问题与潜在影响中等相关
- 范围总体上是适当的，但可行性问题较小
- 贡献是可识别的，但不是开创性的
- 重要性有充分理由

**足够（3）：**
- 研究问题存在但缺乏特异性
- 问题相关性不明确或渐进
- 范围可能太宽或太窄
- 贡献不明确或与现有工作严重重叠
- 重要性论证较弱

**需要改进（2）：**
- 研究问题模糊或定义不明确
- 问题缺乏明确的相关性或重要性
- 范围不适当或不可行
- 贡献没有明确表达
- 没有明确的重要性理由

**差（1）：**
- 没有明确的研究问题
- 问题微不足道或无关紧要
- 范围从根本上来说是有缺陷的
- 没有可识别的贡献
- 没有重要性理由

### 评估清单

- [ ] 研究问题是否明确表述？
- [ ] 这个问题可以用建议的方法来回答吗？
- [ ] 该问题对该领域是否重要？
- [ ] 在资源限制下范围是否可行？
- [ ] 新颖性/贡献是否清楚地表达出来？
- [ ] 是否明确说明了关键假设？
- [ ] 是否定义了成功标准或预期结果？

## 维度2：文献综述

### 质量指标

**优秀（5）：**
- 全面覆盖关键领域的相关文献
- 批判性综合识别模式、矛盾和差距
- 文献是最新的（对于快速发展的领域，大多数文献是过去 3-5 年的文献）
- 来源是权威且经过同行评审的
- 当前工作在学术对话中的明确定位
- 确定工作所解决的真正研究差距

**好（4）：**
- 覆盖良好，关键领域存在微小差距
- 大部分是综合，有一些描述
- 文学大多与一些较旧的基础作品有关
- 来源通常是权威的
- 工作定位已经存在，但可能会更强
- 确定了研究差距，但可能并不重要

**足够（3）：**
- 部分覆盖，有明显差距
- 描述性总结多于综合
- 当前和过时来源的文献组合
- 权威和不太严格的来源的混合
- 现有文献中的定位薄弱
- 研究差距模糊或值得怀疑

**需要改进（2）：**
- 覆盖范围极小，但存在较大差距
- 纯粹描述性的，没有综合性的
- 文学作品基本上已经过时了
- 来源缺乏权威性或严谨性
- 对当前工作的定位很少甚至没有
- 没有发现明显的研究差距

**差（1）：**
- 文献综述不充分或缺失
- 无合成
- 过时或不适当的来源
- 不参与学术对话
- 无间隙识别

### 评估清单

- [ ] 审查是否涵盖所有主要相关领域？
- [ ] 文献是综合性的，而不仅仅是概括性的吗？
- [ ] 来源是否最新且权威？
- [ ] 是否提出了相反的观点？
- [ ] 研究差距是否明确？
- [ ] 当前的工作是否属于现有文献？
- [ ] 引文平衡是否合适（不要过度依赖少数作者）？
- [ ] 是否包含开创性/基础性作品？

### 常见问题

- **覆盖范围不足**：缺少关键论文或研究流
- **描述性列表**：按顺序总结论文而不进行综合
- **过时的来源**：依赖 5-10 年前的文献
- **精挑细选**：仅引用支持假设的工作
- **组织不良**：缺乏主题或概念结构
- **弱间隙识别**：间隙是微不足道的或实际上不是间隙

## 维度 3：方法论和研究设计
### 质量指标

**优秀（5）：**
- 研究设计与研究问题完美契合
- 方法严谨、有效、可靠
- 程序足够详细，可以复制
- 适当的对照、随机化或三角测量
- 承认并减少潜在的偏见
- 全面考虑道德因素
- 明确讨论了限制

**好（4）：**
- 设计适当，有轻微的对齐问题
- 方法合理，有效性问题较小
- 程序大多是可复制的
- 存在一些控制或验证
- 解决了主要偏见
- 提到道德考虑
- 讨论了一些限制

**足够（3）：**
- 设计部分适合问题
- 方法存在显着的有效性问题
- 程序缺乏完整复制的细节
- 有限的控制或验证
- 偏差缓解程度极小
- 道德问题表面化
- 最少讨论的限制

**需要改进（2）：**
- 设计与研究问题不相符
- 方法存在严重的有效性问题
- 程序太模糊，无法复制
- 没有控制或验证
- 偏见未得到解决
- 道德问题未得到解决
- 无限制讨论

**差（1）：**
- 方法不当或缺失
- 方法存在根本缺陷
- 不可复制
- 没有有效性考虑
- 没有道德考虑
- 不承认限制

### 评估清单

- [ ] 方法论适合研究问题吗？
- [ ] 程序描述是否足够详细？
- [ ] 可以从描述中复制该研究吗？
- [ ] 有效性和可靠性是否得到解决？
- [ ] 是否发现并消除了潜在的偏见？
- [ ] 是否讨论了道德考虑？
- [ ] 是否承认限制？
- [ ] 样本量是否合理（对于定量工作）？
- [ ] 定性方法是否严格（如果适用）？

### 设计特定注意事项

**定量研究：**
- 样本量与功效分析
- 对照组和随机化
- 测量的有效性和可靠性
- 统计假设检查

**定性研究：**
- 采样策略和饱和度
- 数据收集程序
- 编码和分析框架
- 可信度标准（可信度、可转移性等）

**混合方法：**
- 整合理由
- 测序理由
- 数据融合策略

## 维度 4：数据收集和来源

### 质量指标

**优秀（5）：**
- 数据来源高度可信且适当
- 样本量充足且合理
- 数据收集程序严谨且系统
- 数据质量控制到位
- 抽样策略确保代表性
- 缺失数据极少且处理得当

**好（4）：**
- 数据来源可靠，问题较少
- 样本量足够
- 收集程序系统化
- 存在一些质量控制
- 抽样合理
- 缺失数据已得到解决

**足够（3）：**
- 数据源是可以接受的，但不是最佳的
- 样本量很小
- 收集程序缺乏严格性
- 有限的质量控制
- 抽样可能存在偏差问题
- 缺失数据处理是基本的

**需要改进（2）：**
- 数据来源存在可信度问题
- 样本量不足
- 收集程序是临时的
- 没有质量控制
- 抽样明显有偏差
- 缺失数据未得到解决

**差（1）：**
- 数据来源不适当或不可靠
- 样本量不足
- 收集不系统
- 没有质量考虑
- 抽样从根本上来说是有缺陷的
- 过多的缺失数据

### 评估清单

- [ ] 数据来源是否可信且适当？
- [ ] 样本量是否足以得出结论？
- [ ] 是否清楚地描述了抽样策略？
- [ ] 样本是否能代表目标人群？
- [ ] 数据收集程序是否系统化？
- [ ] 是否描述了数据质量控制？
- [ ] 缺失数据是否得到解决？
- [ ] 是否讨论了任何潜在的数据偏差？

## 维度 5：分析与解释

### 质量指标

**优秀（5）：**
- 完全适合数据和问题的分析方法
- 使用适当的技术进行严格的分析
- 结果解释合乎逻辑且有充分支持
- 考虑替代解释
- 主张与证据相称
- 假设得到验证
- 分析透明且可重复

**好（4）：**
- 方法适用于小问题
- 分析是合理的
- 解释大多符合逻辑
- 考虑了一些替代方案
- 主张通常与证据相符
- 检查关键假设
- 分析大部分是透明的

**足够（3）：**
- 方法是可以接受的，但不是最佳的
- 分析存在一些技术问题
- 解释存在逻辑漏洞
- 尚未彻底探索替代方案
- 一些说法超出了证据范围
- 假设未得到充分验证
- 分析透明度有限

**需要改进（2）：**
- 方法对数据/问题有疑问
- 分析存在重大技术缺陷
- 翻译支持很差
- 没有其他解释
- 主张远远超过证据
- 未检查假设
- 分析不透明

**差（1）：**
- 方法不恰当
- 分析存在根本性缺陷
- 解释不合逻辑
- 不考虑替代方案
- 没有证据支持的主张
- 没有假设验证
- 分析不透明

### 评估清单

- [ ] 分析方法是否合适？
- [ ] 是否正确应用了统计测试/定性方法？
- [ ] 假设是否得到检验？
- [ ] 解释是否合乎逻辑且得到充分支持？
- [ ] 是否考虑其他解释？
- [ ] 主张与证据强度相符吗？
- [ ] 分析是否可以从描述中重现？
- [ ] 是否承认不确定性？

### 定量分析

- 适当的统计测试
- 检查假设（正态性、同质性等）
- 报告效应大小
- 提供置信区间
- 多次测试修正（如果适用）
- 进行模型诊断

### 定性分析

- 编码框架清晰
- 评估者间的可靠性（如果适用）
- 讨论饱和度
- 检查阴性病例
- 会员检查或验证
- 清晰的审计追踪

## 维度 6：结果和发现

### 质量指标

**优秀（5）：**
- 结果清晰、全面地呈现
- 可视化有效且适当
- 统计或定性的严谨性是显而易见的
- 有效强调主要发现
- 结果直接解决研究问题
- 模式和关系清晰显示
- 报告阴性和无效结果

**好（4）：**
- 结果很清楚，但有一些小的演示问题
- 可视化通常是有效的
- 严谨存在
- 主要发现是可识别的
- 结果主要解决问题
- 显示图案
- 包括一些负面结果

**足够（3）：**
- 结果呈现足够但还可以更清晰
- 可视化是基本的或有问题
- 某些地方的严谨性值得怀疑
- 存在调查结果但未强调
- 与问题部分一致
- 模式不明确
- 阴性结果可以省略

**需要改进（2）：**
- 结果表述不清楚或令人困惑
- 可视化效果不佳或具有误导性
- 缺乏严谨性
- 调查结果难以识别
- 与问题的一致性较差
- 没有清晰的模式
- 仅显示积极结果

**差（1）：**
- 结果呈现不佳或缺失
- 可视化不恰当或缺失
- 没有严格的证据
- 研究结果尚不明确
- 结果并不能解决问题
- 没有可识别的模式
- 结果显得有选择性

### 评估清单

- [ ] 结果是否清楚地呈现？
- [ ] 结果是否直接解决研究问题？
- [ ] 可视化是否适当且有效？
- [ ] 是否强调了主要发现？
- [ ] 是否报告阴性/无效结果？
- [ ] 是否报告了适当的精度（p 值、CI、效应量）？
- [ ] 定性研究结果是否有数据摘录支持？
- [ ] 是否有选择性报告的证据？

### 演示质量

**表格：**
- 清晰的标签和标题
- 适当的精度
- 有逻辑地组织
- 不太复杂

**数字：**
- 清晰的轴和图例
- 适当的图表类型
- 专业的外观
- 无障碍（色盲友好）

**文字：**
- 强调主要发现
- 避免表格/数字的冗余
- 使用适当的统计语言

## 维度 7：学术写作和演讲

### 质量指标

**优秀（5）：**
- 写作清晰、简洁、准确
- 组织逻辑清晰，流程流畅
- 学术语气恰当且一致
- 语法和机制完美无缺
- 技术术语的使用正确
- 适合目标受众
- 摘要/总结全面、准确

**好（4）：**
- 文字清晰，有轻微的尴尬
- 组织逻辑清晰，流程良好
- 语气最合适
- 很少有语法/机械错误
- 技术术语大多正确
- 一般可访问
- 摘要足够

**足够（3）：**
- 文字可以理解，但存在清晰度问题
- 组织存在一些逻辑缺陷
- 语气不一致
- 明显的语法/机械错误
- 一些技术术语的误用
- 目标受众的可访问性问题
- 摘要不完整或模糊

**需要改进（2）：**
- 写作常常不清楚或冗长
- 组织和流程较差
- 语气不恰当
- 频繁的语法/机械错误
- 技术术语问题
- 目标受众无法访问
- 摘要较差或缺失

**差（1）：**
- 文字不清晰且难以理解
- 没有明确的组织
- 语气不恰当
- 普遍存在的语法/机械错误
- 技术术语不正确
- 无法访问
- 没有足够的摘要

### 评估清单

- [ ] 书写是否清晰、简洁？
- [ ] 组织是否符合逻辑？
- [ ] 语气适合学术写作吗？
- [ ] 语法和结构是否正确？
- [ ] 技术术语使用得当吗？
- [ ] 必要时是否解释行话？
- [ ] 摘要是否准确地总结了作品？
- [ ] 各部分之间的过渡是否平滑？
- [ ] 目标受众是否明确？

### 常见写作问题

- **冗长**：不必要的复杂或冗长的散文
- **过度使用被动语态**：降低清晰度和直接性
- **段落结构**：缺乏主题句或连贯性
- **冗余**：不必要地重复信息
- **逻辑流程**：想法之间的转换不佳
- **精确**：含糊或模棱两可的语言
- **辅助功能**：技术性太强或技术性不够

## 维度 8：引文和参考文献

### 质量指标

**优秀（5）：**
- 所有主张均得到适当引用
- 来源是权威且最新的
- 引文准确且完整
- 代表不同的观点
- 引文格式一致且正确
- 自引与他人引用之间的平衡
- 正确使用主要来源

**好（4）：**
- 大多数主张都被引用
- 来源通常是权威的
- 引用错误很少
- 资源的合理多样性
- 格式基本一致
- 引文平衡良好
- 主要和次要来源的混合

**足够（3）：**
- 一些主张缺乏引用
- 来源质量参差不齐
- 几个引用错误
- 来源多样性有限
- 格式不一致
- 引文平衡问题
- 过度依赖二手资源

**需要改进（2）：**
- 许多主张未引用
- 来源有问题
- 大量引用错误
- 来源基础狭窄
- 格式不一致
- 过多的自引或狭义引用
- 不适当的来源（例如，仅次要来源）

**差（1）：**
- 引用不足
- 来源不可靠
- 普遍存在的引用错误
- 最小的来源多样性
- 没有一致的格式
- 严重的引用失衡
- 不适当的来源类型

### 评估清单

- [ ] 是否引用了所有事实主张？
- [ ] 是否在适当的时候引用了主要来源？
- [ ] 来源是否权威且经过同行评审？
- [ ] 引用的观点是否平衡？
- [ ] 引文是否准确（作者、日期、页数）？
- [ ] 引文格式是否一致？
- [ ] 自引是否合适（通常<20%）？
- [ ] 来源是否最新（对于时间敏感的主题）？
- [ ] 是否包含相关的经典/开创性作品？

### 引文质量评估

**来源类型（按大多数学术作品的优先顺序）：**
1. 同行评审的期刊文章
2. 知名出版社的学术书籍
3. 会议记录（视场而定）
4. 知名机构的技术报告
5. 论文/论文
6. 预印本（谨慎，取决于领域）
7.灰色文献（限制使用）
8. 网站（很少适用，事实数据除外）

**危险信号：**
- 维基百科作为主要来源
- 过多的自引（>30%）
- 只引用支持假设的论文
- 当前来源存在时过时的来源
- 缺少该领域的关键论文
- 仅在全文可用时引用摘要
- 引用格式不一致或不正确

## 横切考虑因素

### 再现性

跨维度评估：
- 方法是否足够详细以供复制？
- 数据和代码是否可用（或解释了可用性）？
- 分析步骤是否透明？
- 是否指定了材料/仪器？

### 道德规范

考虑：
- IRB 批准（针对人类受试者）
- 知情同意
- 隐私和保密
- 利益冲突
- 研究诚信
- 数据共享道德

### 偏见和局限性

评估是否：
- 承认潜在的偏见
- 诚实地讨论局限性
- 指定边界条件
- 适当地声明了普遍性

### 影响和意义

考虑：
- 理论贡献
- 实际影响
- 政策相关性
- 方法论创新
- 领域进步

## 评分指南

### 维度权重（建议，根据上下文调整）

- 问题表述：15%
- 文献综述：15%
- 方法论：20%
- 数据收集：10%
- 分析：15%
- 结果：10%
- 写作：10%
- 引用率：5%

### 总体评估阈值

- **例外 (4.5-5.0)**：准备顶级出版物
- **强 (4.0-4.4)**：经过细微修改即可发布
- **好（3.5-3.9）**：需要重大修改，有前途的工作
- **可接受 (3.0-3.4)**：需要进行重大修改
- **弱（2.0-2.9）**：基本问题，需要重大返工
- **差（<2.0）**：未经完整修订不适合出版

### 上下文调整

根据以下因素调整标准：
- **阶段**：提案 < 草案 < 最终提交
- **地点**：学生论文 < 会议 < 期刊 < 顶级期刊
- **类型**：理论 < 经验 < 荟萃分析
- **领域**：标准因学科而异
- **目的**：教育 < 专业 < 出版

1. **开始评估之前仔细阅读作品**
