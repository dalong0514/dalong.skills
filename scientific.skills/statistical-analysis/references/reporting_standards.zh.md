# 统计报告标准

本文件提供了根据 APA（美国心理学会）风格和学术出版物的一般最佳实践报告统计分析的指南。

## 一般原则

1. **透明度**：报告足够的详细信息以供复制
2. **完整性**：包括所有计划的分析和结果
3. **诚实**：报告非重大发现和违规行为
4. **清晰度**：为您的受众写作，定义技术术语
5. **再现性**：尽可能提供代码、数据或补充

---

## 预注册和规划

### 报告内容（最好是在数据收集之前）

1. **假设**：明确陈述，适当时具有方向性
2. **样本量理由**：功效分析或其他理由
3. **数据收集停止规则**：什么时候停止收集数据？
4. **变量**：收集的所有变量（不仅仅是分析的变量）
5. **排除标准**：排除参与者/数据点的规则
6. **统计分析**：计划的测试，包括：
   - 初步分析
   - 二次分析
   - 探索性分析（如此标记）
   - 缺失数据的处理
   - 多重比较修正
   - 假设检查

**为什么要预先注册？**
- 防止HARKing（已知结果后进行假设）
- 区分验证性分析和探索性分析
- 提高可信度和再现性

**平台**：OSF、AsPredicted、ClinicalTrials.gov

## 方法部分

### 参与者

**报告内容**：
- 总计 N，包括排除的参与者
- 相关人口统计数据（年龄、性别等）
- 招聘方式
- 纳入/排除标准
- 自然减员/退出

**示例**：
> “参与者是从心理学课程中招募来换取课程学分的 150 名本科生（98 名女性，52 名男性；M_age = 19.4 岁，SD = 1.2，范围 18-24）。由于数据不完整 (n = 3) 或未能通过注意力检查 (n = 2)，有 5 名参与者被排除在外，最终样本为 145 名。”

### 设计

**报告内容**：
- 研究设计（科目间、科目内、混合）
- 自变量和水平
- 因变量
- 控制变量/协变量
- 随机化程序
- 盲法（单盲、双盲）

**示例**：
> “使用了 2（反馈：积极与消极）× 2（时间：立即与延迟）受试者间析因设计。使用计算机生成的随机序列将参与者随机分配到条件。主要结果是以正确反应的数量（0-20 等级）衡量的任务绩效。”

### 措施

**报告内容**：
- 措施/工具的全名
- 物品数量
- 量表/响应格式
- 评分方法
- 可靠性（Cronbach's α、ICC 等）
- 有效性证据（如果适用）

**示例**：
> “使用贝克抑郁量表-II（BDI-II；Beck 等人，1996）评估抑郁症，这是一项 21 项自我报告测量，按 4 分制 (0-3) 进行评分。总分范围从 0 到 63，分数越高表明抑郁严重程度越高。BDI-II 在该样本中表现出极好的内部一致性 (α = .91)。”

### 程序

**报告内容**：
- 逐步描述参与者所做的事情
- 时间和持续时间
- 给出的指示
- 任何操纵或干预

**示例**：
>“参与者通过 Qualtrics 在线完成了研究。在提供知情同意书后，他们完成了人口统计问题，被随机分配到四种条件之一，完成了实验任务（大约 15 分钟），并完成了结果测量和汇报。整个会议持续了大约 30 分钟。”

### 数据分析

**报告内容**：
- 使用的软件（含版本）
- 显着性水平 (α)
- 测试尾部（单尾或双尾）
- 进行假设检查
- 缺失数据处理
- 异常值处理
- 多重比较修正
- 使用的效应量测量

**示例**：
> “所有分析均使用 Python 3.10、scipy 1.11 和 statsmodels 0.14 进行。所有显着性检验均使用 0.05 的 alpha 水平。分别使用 Shapiro-Wilk 和 Levene 检验评估正态性和方差同质性的假设。使用列表删除处理缺失数据（所有变量 < 2%）。对平均值超过 3 SD 的异常值进行缩尾处理。对于主要数据方差分析、部分 eta 平方 (η²_p) 被报告为效应大小测量，使用 Tukey 的 HSD 来控制家庭错误率。”

## 结果部分

### 描述性统计

**报告内容**：
- 样本量（每组，如果适用）
- 集中趋势测量（M、Mdn）
- 变异性测量（SD、IQR、范围）
- 置信区间（适当时）

**示例（连续结果）**：
> “A 组 (n = 48) 的平均得分为 75.2 (SD = 8.5, 95% CI [72.7, 77.7])，而 B 组 (n = 52) 的平均得分为 68.3 (SD = 9.2, 95% CI [65.7, 70.9])。”

**示例（分类结果）**：
> “在 145 名参与者中，89 人 (61.4%) 选择了选项 A，42 人 (29.0%) 选择了选项 B，14 人 (9.7%) 选择了选项 C。”

**描述性统计表**：
- 使用表格表示多个变量或组
- 包括 M、SD 和 n（最小值）
- 可以包括范围、偏度、峰度（如果相关）

---

### 假设检查

**报告内容**：
- 测试了哪些假设
- 诊断测试结果
- 假设是否得到满足
- 如有违反将采取的行动

**示例**：
> “使用 Shapiro-Wilk 检验评估正态性。A 组（W = 0.97，p = .18）和 B 组（W = 0.96，p = .12）的数据没有显着偏离正态性。Levene 检验表明方差同质性，F(1, 98) = 1.23，p = .27。因此，满足独立样本 t 检验的假设。”

**示例（违规）**：
> “Shapiro-Wilk 检验表明 C 组显着偏离正态性（W = 0.89，p = .003）。因此，使用非参数 Mann-Whitney U 检验代替独立样本 t 检验。”

---

### 推论统计

#### T 检验

**报告内容**：
- 检验统计量（t）
- 自由度
- p 值（如果 p > .001，则精确，否则 p < .001）
- 效应量（Cohen's d 或 Hedges' g）与 CI
- 作用方向
- 测试是单尾还是双尾

**格式**：t(df) = 值，p = 值，d = 值，95% CI [下限，上限]

**示例（独立 t 检验）**：
> “A 组 (M = 75.2，SD = 8.5) 得分显着高于 B 组 (M = 68.3，SD = 9.2)，t(98) = 3.82，p < .001，d = 0.77，95% CI [0.36, 1.18]，双尾。”

**示例（配对 t 检验）**：
> “从预测（M = 65.4，SD = 10.2）到后测（M = 71.8，SD = 9.7），分数显着增加，t(49) = 4.21，p < .001，d = 0.64，95% CI [0.33, 0.95]。”

**示例（韦尔奇 t 检验）**：
> “由于方差不等，使用了韦尔奇 t 检验。A 组得分显着高于 B 组，t(94.3) = 3.65，p < .001，d = 0.74。”

**示例（不重要）**：
> “A 组（M = 72.1，SD = 8.3）和 B 组（M = 70.5，SD = 8.9）之间没有显着差异，t(98) = 0.91，p = .36，d = 0.18，95% CI [-0.21, 0.57]。”

---

#### 方差分析

**报告内容**：
-F统计量
- 自由度（效果、误差）
- p 值
- 效应大小（η²、η²_p 或 ω²）
- 所有组的平均值和标准差
- 事后测试结果（如果重要）

**格式**：F(df_effect, df_error) = 值，p = 值，η²_p = 值

**示例（单向方差分析）**：
> “治疗条件对测试分数有显着的主效应，F(2, 147) = 8.45，p < .001，η²_p = .10。使用 Tukey HSD 进行事后比较显示，条件 A (M = 78.2，SD = 7.3) 的得分显着高于条件 B (M = 71.5，SD = 8.1，p = .002，d = 0.87）和条件 C（M = 70.1，SD = 7.9，p < .001，d = 1.07）。条件 B 和 C 没有显着差异（p = .52，d = 0.18）。

**示例（阶乘方差分析）**：
> “A 2（反馈：正面与负面）× 2（时间：立即与延迟）受试者间方差分析显示反馈具有显着主效应，F(1, 146) = 12.34，p < .001，η²_p = .08，但时间没有显着主效应，F(1, 146) = 2.10，p = .15，η²_p =重要的是，交互作用显着，F(1, 146) = 6.78，p = .01，η²_p = .04。简单效应分析表明，正反馈改善了即时计时的性能（M_diff = 8.2，p < .001），但没有改善延迟计时的性能（M_diff = 1.3，p = .42）。

**示例（重复测量方差分析）**：
> “单向重复测量方差分析显示时间点对焦虑评分有显着影响，F(2, 98) = 15.67，p < .001，η²_p = .24。Mauchly 检验表明违反了球形假设，χ²(2) = 8.45，p = .01，因此报告了 Greenhouse-Geisser 校正值 (ε = 0.87）与 Bonferroni 校正的成对比较显示......”

---

#### 相关性

**报告内容**：
- 相关系数（r 或 ρ）
- 样本量
- p 值
- 方向和强度
- 置信区间
- 决定系数 (r²)（如果相关）

**格式**：r(df) = 值，p = 值，95% CI [下限，上限]

**示例（皮尔逊）**：
> “学习时间和考试成绩之间存在中度正相关，r(148) = .42，p < .001，95% CI [.27, .55]，表明考试成绩的 18% 方差与学习时间共享 (r² = .18)。”

**示例（斯皮尔曼）**：
> “Spearman 排名顺序相关性显示班级排名和动机之间存在显着的正相关关系，ρ(118) = .38，p < .001，95% CI [.21, .52]。”

**示例（不重要）**：
> “年龄和反应时间之间没有显着相关性，r(98) = -.12，p = .23，95% CI [-.31, .08]。”

---

#### 回归

**报告内容**：
- 整体模型拟合（R²、调整后的 R²、F 检验）
- 每个预测变量的系数（B、SE、β、t、p）
- 效果大小
- 系数的置信区间
- 方差膨胀因子（如果评估多重共线性）

**格式**：B = 值、SE = 值、β = 值、t = 值、p = 值、95% CI [下限、上限]

**示例（简单回归）**：
> “简单线性回归显示，学习时间可显着预测考试成绩，F(1, 148) = 42.5，p < .001，R² = .22。具体来说，每增加一小时的学习时间，考试成绩就会增加 2.4 分（B = 2.40，SE = 0.37，β = .47，t = 6.52，p < .001，95% CI [1.67, 3.13])。”

**示例（多元回归）**：
> “进行多元线性回归，根据学习时间、之前的 GPA 和出勤率来预测考试成绩。总体模型显着，F(3, 146) = 45.2，p < .001，R² = .48，调整后的 R² = .47。学习时间 (B = 1.80，SE = 0.31，β = .35，t = 5.78，p < .001， 95% CI [1.18, 2.42]）和之前的 GPA（B = 8.52，SE = 1.95，β = .28，t = 4.37，p < .001，95% CI [4.66，12.38]）是显着的预测因素，但出勤率不是（B = 0.15，SE = 0.12，β = .08， t = 1.25，p = .21，95% CI [-0.09, 0.39]）。多重共线性不是问题，因为所有 VIF 值均低于 1.5。”

**示例（逻辑回归）**：
> “进行逻辑回归以根据学习时间预测通过/失败状态。整体模型显着，χ2(1) = 28.7，p < .001，Nagelkerke R2 = .31。每增加一个学习小时，通过的几率就会增加 1.35 倍（OR = 1.35，95% CI [1.18, 1.54]，p < .001）。模型正确分类76% 的病例（敏感性 = 81%，特异性 = 68%）。”

#### 卡方检验

**报告内容**：
- χ2 统计量
- 自由度
- p 值
- 效应大小（Cramér 的 V 或 φ）
- 观察到的和预期的频率（或百分比）

**格式**：χ2(df, N = 总计) = 值，p = 值，Cramér 的 V = 值

**示例 (2×2)**：
> “卡方独立性检验显示治疗组与结果之间存在显着相关性，χ²(1, N = 150) = 8.45，p = .004，φ = .24。具体而言，治疗组中 72% 的参与者有所改善，而对照组中只有 48% 的参与者有所改善。”

**示例（较大的表格）**：
> “卡方检验检验了教育水平（高中、学士、研究生）与政治倾向（自由派、温和派、保守派）之间的关系。这种关联非常显着，χ2(4, N = 300) = 18.7，p = .001，Cramér V = .18，表明存在小到中等关联。”

**示例（费舍尔精确）**：
> “由于预期细胞计数低于 5，因此使用了 Fisher 精确检验。治疗与结果之间的相关性显着，p = 0.018（双尾），OR = 3.42，95% CI [1.21, 9.64]。”

#### 非参数检验

**曼惠特尼大学**：
> “Mann-Whitney U 检验表明，A 组（Mdn = 75，IQR = 10）的得分显着高于 B 组（Mdn = 68，IQR = 12），U = 845，z = 3.21，p = .001，r = .32。”

**Wilcoxon 符号秩**：
> “Wilcoxon 符号秩检验显示，从预测（Mdn = 65，IQR = 15）到后测（Mdn = 72，IQR = 14），分数显着增加，z = 3.89，p < .001，r = .39。”

**克鲁斯卡尔-沃利斯**：
> “Kruskal-Wallis 检验显示三种条件之间存在显着差异，H(2) = 15.7，p < .001，η² = .09。与 Bonferroni 校正的后续成对比较显示......”

#### 贝叶斯统计

**报告内容**：
- 使用先前的发行版
- 后验估计（平均值/中位数、可信区间）
- 贝叶斯因子（如果假设检验）
- 收敛诊断（R-hat、ESS）
- 事后预测检查

**示例（贝叶斯 t 检验）**：
> “使用信息量较弱的先验（均值差为正态 (0, 1)）进行贝叶斯独立样本 t 检验。均值差的后验分布平均值为 6.8（95% 可信区间 [3.2, 10.4]），表明 A 组得分高于 B 组。贝叶斯因子 BF₁₀ = 45.3 为组间差异提供了非常有力的证据。 A 组平均值超过 B 组平均值的后验概率为 99.8%。”

**示例（贝叶斯回归）**：
> “贝叶斯线性回归拟合了信息量较弱的先验（系数为正态 (0, 10)，残差 SD 为 Half-Cauchy(0, 5)）。模型显示学习时间可靠地预测了考试成绩（β = 0.52，95% CI [0.38, 0.66]；0 不包含在区间内）。所有收敛诊断均令人满意（R-hat < 1.01，ESS >所有参数均为 1000）。后验预测检查表明模型拟合良好。”

## 效果大小

### 始终报告

**为什么**：
- p 值并不表示大小
- APA 和大多数期刊都要求
- 荟萃分析必不可少
- 传达实际意义

**效果大小是多少？**
- T 检验：Cohen's d 或 Hedges' g
- 方差分析：η²、η²_p 或 ω²
- 相关性：r（已经是效果大小）
- 回归：β（标准化）、R²、f²
- 卡方：Cramér 的 V 或 φ

**具有置信区间**：
- 尽可能报告效应量的 CI
- 显示估计的精度
- 比单独的点估计提供更多信息

## 图表

### 何时使用表格与图表

**表格**：
- 需要精确的值
- 许多变量/条件
- 描述性统计
- 回归系数
- 相关矩阵

**数字**：
- 模式和趋势
- 发行版
- 互动
- 组间比较
- 时间序列

### 图形指南

**一般**：
- 清晰易读的标签
- 足够的字体大小（≥ 10pt）
- 高分辨率（出版物≥ 300 dpi）
- 单色友好（色盲可访问）
- 误差线（SE 或 95% CI；指定哪个！）
- 需要时的图例

**常见图形类型**：
- 条形图：组比较（包括误差线）
- 箱形图：分布、异常值
- 散点图：相关性、关系
- 折线图：随时间变化、交互
- 小提琴图：分布（比箱线图更好）

**示例图标题**：
> “图 1. 按学习条件划分的平均考试成绩。误差线代表 95% 置信区间。* p < .05，** p < .01，*** p < .001。”

### 表格指南

**一般**：
- 清除列和行标签
- 一致的小数位数（通常为 2 位）
- 仅水平线（非垂直线）
- 表下方注释以进行澄清
- 斜体统计符号（p、M、SD、F、t、r）

## 要避免的常见错误

1. **报告 p = .000**：改为报告 p < .001
2. **省略效果大小**：始终包含它们
3. **不报告假设检查**：描述测试和结果
4. **令人困惑的统计意义和实际意义**：讨论两者
5. **仅报告重要结果**：报告所有计划的分析
6. **使用“证明”或“确认”**：使用“支持”或“一致”
7. **对于 0.05 < p < .10** 说“边际显着”：显着或不显着
8. **仅报告 p 值的一位小数**：使用两位（p = .03，而不是 p = .0）
9. **未指定一尾与二尾**：始终澄清
10. **不一致的舍入**：始终保持一致

## 空结果

### 如何报告非重大发现

**别说**：
- “没有效果”
- “X 和 Y 不相关”
- “群体是等价的”

**请说**：
- “没有显着差异”
- “效果没有统计学意义”
- “我们没有找到关系的证据”

**包括**：
- 精确的 p 值（不仅仅是“ns”或“p > .05”）
- 效果大小（即使不显着也显示大小）
- 置信区间（可能包括有意义的值）
- 功效分析（研究是否有足够的功效？）

**示例**：
> “与我们的假设相反，音乐（M = 72.1，SD = 8.3）和沉默（M = 70.5，SD = 8.9）条件下的创造力得分没有显着差异，t(98) = 0.91，p = 0.36，d = 0.18，95% CI [-0.21, 0.57]。事后敏感性分析显示，该研究80% 的功效可检测 d = 0.57 或更大的效应，这表明零发现可能反映出检测小效应的功效不足。”

## 再现性

### 分享材料

1. **数据**：去识别化的原始数据（如果敏感则聚合）
2. **代码**：分析脚本
3. **材料**：刺激、措施、协议
4. **补充**：附加分析、表格

**分享地点**：
- 开放科学框架（OSF）
- GitHub（代码）
- 期刊增刊
- 机构知识库

**纸质版**：
> "数据、分析代码和材料可在 https://osf.io/xxxxx/" 获取
