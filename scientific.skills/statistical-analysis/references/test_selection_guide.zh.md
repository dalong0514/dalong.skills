# 统计测试选择指南

本指南提供了一个决策树，用于根据研究问题、数据类型和研究设计选择适当的统计检验。

## 测试选择的决策树

### 1. 比较组

#### 两个独立的组
- **连续结果，正态分布**：独立样本 t 检验
- **连续结果，非正常**：Mann-Whitney U 检验（Wilcoxon 秩和检验）
- **二元结果**：卡方检验或 Fisher 精确检验（如果预期计数 < 5）
- **顺序结果**：Mann-Whitney U 检验

#### 两个配对/依赖组
- **连续结果，正态分布**：配对 t 检验
- **连续结果，非正常**：Wilcoxon 符号秩检验
- **二元结果**：麦克尼马尔检验
- **顺序结果**：Wilcoxon 符号秩检验

#### 三个或更多独立组
- **连续结果，正态分布，等方差**：单向方差分析
- **连续结果，正态分布，不等方差**：韦尔奇方差分析
- **连续结果，非正常**：Kruskal-Wallis H 检验
- **二元/分类结果**：卡方检验
- **顺序结果**：Kruskal-Wallis H 检验

#### 三个或更多配对/相关组
- **连续结果，正态分布**：重复测量方差分析
- **连续结果，非正常**：弗里德曼检验
- **二元结果**：Cochran 的 Q 检验

#### 多重因素（因子设计）
- **连续结果**：双向方差分析（或更高级方差分析）
- **有协变量**：ANCOVA
- **因素内部和因素之间的混合**：混合方差分析

### 2. 变量之间的关系

#### 两个连续变量
- **线性关系，二元正态**：皮尔逊相关
- **单调关系或非正态**：斯皮尔曼等级相关
- **基于排名的数据**：Spearman 或 Kendall 的 tau

#### 一种连续结果，一个或多个预测因素
- **单个连续预测器**：简单线性回归
- **多个连续/分类预测变量**：多元线性回归
- **分类预测变量**：ANOVA/ANCOVA 框架
- **非线性关系**：多项式回归或广义加性模型 (GAM)

#### 二元结果
- **单一预测器**：逻辑回归
- **多个预测变量**：多重逻辑回归
- **罕见事件**：精确逻辑回归或 Firth 方法

#### 计算结果
- **泊松分布**：泊松回归
- **过度分散计数**：负二项式回归
- **零膨胀**：零膨胀泊松/负二项式

#### 事件发生时间结果
- **比较生存曲线**：对数秩检验
- **协变量建模**：Cox 比例风险回归
- **参数生存模型**：威布尔、指数、对数正态

### 3.协议和可靠性

#### 评估者间的可靠性
- **分类评级，2 个评级者**：Cohen's kappa
- **分类评级，>2 个评级者**：Fleiss' kappa 或 Krippendorff's alpha
- **连续评级**：组内相关系数 (ICC)

#### 重测可靠性
- **连续测量**：ICC 或 Pearson 相关性
- **内部一致性**：Cronbach's alpha

#### 方法之间的协议
- **连续测量**：Bland-Altman 分析
- **类别分类**：Cohen's kappa

### 4. 分类数据分析

#### 列联表
- **2x2 表**：卡方检验或 Fisher 精确检验
- **大于 2x2**：卡方检验
- **有序类别**：Cochran-Armitage 趋势测试
- **配对类别**：McNemar 测试 (2x2) 或 McNemar-Bowker 测试（较大）

### 5.贝叶斯替代方案

上述任何测试都可以使用贝叶斯方法进行：
- **组比较**：贝叶斯 t 检验、贝叶斯方差分析
- **相关性**：贝叶斯相关性
- **回归**：贝叶斯线性/逻辑回归

**贝叶斯方法的优点：**
- 提供给定数据的假设概率
- 自然地融入先验信息
- 提供可信区间而不是置信区间
- 没有 p 值解释问题

## 关键考虑因素

### 样本量
- 小样本 (n < 30)：考虑非参数检验或精确方法
- 样本量非常大：即使很小的影响也可能具有统计显着性；关注效应大小

### 多重比较
- 进行多项测试时，使用以下方法调整多重比较：
  - Bonferroni 修正（保守）
  - Holm-Bonferroni（不太保守）
  - 错误发现率 (FDR) 控制 (Benjamini-Hochberg)
  - Tukey HSD 用于事后方差分析比较
### 缺失数据
- 完整的案例分析（列表删除）
- 多重插补
- 最大似然法
- 确保理解缺失数据机制（MCAR、MAR、MNAR）

### 效果大小
- 始终报告效应大小以及 p 值
- 请参阅 `effect_sizes_and_power.md` 获取指导

### 研究设计注意事项
- 随机对照试验：标准参数/非参数测试
- 观察性研究：考虑混杂并使用回归/匹配
- 聚类/嵌套数据：使用混合效应模型或 GEE
- 时间序列：使用时间序列方法（ARIMA等）
