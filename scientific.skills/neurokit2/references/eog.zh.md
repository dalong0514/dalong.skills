# 眼电图 (EOG) 分析

## 概述

眼电图 (EOG) 通过检测眼睛位置变化产生的电位差来测量眼球运动和眨眼。 EOG 用于睡眠研究、注意力研究、阅读分析和 EEG 伪影校正。

## 主要加工流程

### eog_process()

自动化 EOG 信号处理管道。

```python
signals, info = nk.eog_process(eog_signal, sampling_rate=500, method='neurokit')
```

**管道步骤：**
1.信号清洗（过滤）
2. 眨眼检测
3. 眨眼率计算

**退货：**
- `signals`：数据帧具有：
  - `EOG_Clean`：过滤后的 EOG 信号
  - `EOG_Blinks`：二进制闪烁标记 (0/1)
  - `EOG_Rate`：瞬时眨眼率（眨眼/分钟）
- `info`：带有闪烁索引和参数的字典

**方法：**
- `'neurokit'`：NeuroKit2 优化方法（默认）
- `'agarwal2019'`：Agarwal 等人。 (2019) 算法
- `'mne'`：MNE-Python 方法
- `'brainstorm'`：头脑风暴工具箱方法
- `'kong1998'`：Kong 等人。 （1998）方法

## 实际考虑

### 采样率建议
- **最小值**：100 Hz（基本眨眼检测）
- **标准**：250-500 Hz（研究应用）
- **高分辨率**：1000 Hz（详细波形分析、眼跳）
- **睡眠研究**：200-250 Hz（典型值）

### 录音时长
- **眨眼检测**：任何持续时间（≥1 次眨眼）
- **眨眼率估计**：稳定估计≥60秒
- **与事件相关**：取决于范例（每次试验的秒数）
- **睡眠 EOG**：小时（整晚）

### 电极放置

**标准配置：**

**水平眼电图 (HEOG)：**
- 两个电极：左眼和右眼的外眼角
- 测量水平眼球运动（眼跳、平滑追踪）
- 双极录音（左 - 右）

**垂直眼电图 (VEOG)：**
- 两个电极：一只眼睛的上方和下方（通常是右眼）
- 测量垂直眼球运动和眨眼
- 双极录音（上 - 下）

**睡眠眼电图：**
- 经常使用略有不同的位置（寺庙区域）
- E1：左眼外眦外侧1厘米、下1厘米
- E2：右眼外眦外侧1厘米、上1厘米
- 捕捉水平和垂直运动

**脑电图污染去除：**
- 额电极（Fp1、Fp2）可以作为 EOG 代理
- EEG 预处理中常见的基于 ICA 的 EOG 伪影去除

### 常见问题和解决方案

**电极问题：**
- 接触不良：振幅低，噪音大
- 皮肤准备：清洁、轻微擦伤
- 导电胶：确保接触良好

**文物：**
- 肌肉活动（尤其是额肌）：高频噪音
- 运动：电缆伪影、头部运动
- 电噪声：50/60 Hz 嗡嗡声（正确接地）

**饱和度：**
- 较大的眼跳可能会使放大器饱和
- 调整增益或电压范围
- 在低分辨率系统中更常见

## 临床和研究应用

**脑电图伪影校正：**
- 眨眼会污染额叶脑电图通道
- ICA 或回归方法消除 EOG 伪影
- ERP学习必备

**睡眠分期：**
- 快速眼动睡眠期间的快速眼动 (REM)
- 困倦时眼球缓慢转动
- 睡眠开始和阶段转换

**注意力和认知负荷：**
- 在要求较高的任务期间抑制眨眼频率
- 在任务边界处闪烁集群（自然断点）
- 自发眨眼作为注意力转移的指标

**疲劳和困倦监测：**
- 困倦时眨眼时间增加
- 眼睑闭合速度较慢
- 部分或不完整的眨眼
- 驾驶员监控应用程序

**阅读和视觉处理：**
- 阅读时禁止眨眼
- 扫视期间的眼球运动（线变化）
- 疲劳影响阅读效率

**神经系统疾病：**
- **帕金森病**：自发眨眼率降低
- **精神分裂症**：眨眼频率增加
- **抽动秽语综合征**：过度眨眼（抽动）
- **干眼综合症**：眨眼次数增多、不完全

**情感和社会认知：**
- 社交互动中的眨眼同步
- 眨眼频率的情绪调节
- ERP 中与眨眼相关的潜力

**人机交互：**
- 视线追踪预处理
- 注意力监控
- 用户参与度评估

## 可通过 EOG 检测的眼动类型

**闪烁：**
- 振幅大，持续时间短（100-400 ms）
- NeuroKit2 主要焦点
- 垂直眼电图最敏感

**眼跳：**
- 快速、弹道式眼球运动（20-80 毫秒）
- 阶梯状电压偏转
- 水平或垂直
- 需要更高的采样率进行详细分析

**顺利追击：**
- 缓慢跟踪移动物体
- 类似斜坡的电压变化
- 振幅低于眼跳

**固定装置：**
- 目光稳定
- 基线 EOG 有小波动
- 持续时间各不相同（读取时通常为 200-600 毫秒）

**注意：** 详细的扫视/注视分析通常需要眼动追踪（红外、基于视频）。眼电图对于眨眼和粗眼球运动很有用。

## 解释指南

**眨眼率：**
- **正常休息**：15-20 次眨眼/分钟
- **<10 次眨眼/分钟**：视觉任务参与度、注意力
- **>30 次眨眼/分钟**：压力、眼睛干涩、疲劳
- **取决于上下文**：任务要求、照明、屏幕使用

**眨眼持续时间：**
- **正常**：100-400 毫秒（平均约 200 毫秒）
- **延长**：困倦、疲劳（>500 毫秒）
- **短**：正常警觉性

**眨眼幅度：**
- 因电极放置位置和个人而异
- 受试者内比较最可靠
- 眨眼不完全：幅度减小（干眼、疲劳）

**时间模式：**
- **簇状眨眼**：任务或认知状态之间的转换
- **抑制眨眼**：主动视觉处理，持续注意力
- **刺激后眨眼**：完成视觉处理后
